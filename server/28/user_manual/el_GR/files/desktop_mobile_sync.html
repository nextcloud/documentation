Συγχρονισμός επιτραπέζιων υπολογιστών και κινητών

Για συγχρονισμό αρχείων με τον επιτραπέζιο υπολογιστή σας, σας συνιστούμε να χρησιμοποιήσετε το Nextcloud Sync Client για Windows, macOS και Linux.

Ο πελάτης Nextcloud Desktop Sync σάς δίνει τη δυνατότητα να συνδεθείτε στον διακομιστή Nextcloud. Μπορείτε να δημιουργήσετε φακέλους στον αρχικό σας κατάλογο και να διατηρήσετε το περιεχόμενο αυτών των φακέλων συγχρονισμένο με τον διακομιστή Nextcloud. Απλώς αντιγράψτε ένα αρχείο στον κατάλογο και ο υπολογιστής-πελάτης Nextcloud κάνει τα υπόλοιπα. Πραγματοποιήστε μια αλλαγή στα αρχεία σε έναν υπολογιστή, θα ρέει στους άλλους χρησιμοποιώντας αυτούς τους υπολογιστές-πελάτες συγχρονισμού. Θα έχετε πάντα τα πιο πρόσφατα αρχεία σας όπου κι αν βρίσκεστε.

Η χρήση του τεκμηριώνεται ξεχωριστά στο Nextcloud Desktop Client Manual.

Πελάτες κινητής τηλεφωνίας

Επισκεφθείτε την προσωπική σας σελίδα στη διεπαφή Nextcloud Web για να βρείτε συνδέσμους λήψης για πελάτες συγχρονισμού για κινητά Android και iOS. Εναλλακτικά, επισκεφθείτε τη σελίδα «Λήψη Nextcloud <https://nextcloud.com/install/>» _.

Unfortunately, 2-way-sync, a.k.a bidirectional sync, isn’t fully implemented in the Android client yet. See this issue on GitHub for the current feature’s status.
