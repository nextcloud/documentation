桌面電腦和移動設備同步

我們建議使用適用於 Windows、macOS 和 Linux 的 Nextcloud Sync Client 來與您的桌面電腦同步檔案。

Nextcloud 桌面同步客戶端使您能夠連接到您的 Nextcloud 伺服器。您可以在您的主目錄中創建資料夾，並將這些資料夾的內容與您的 Nextcloud 伺服器同步。只需將檔案複製到目錄中，Nextcloud 桌面客戶端即可完成剩下的工作。對一部電腦上的檔案進行更改，它會使用這些桌面同步客戶端在其他電腦上傳輸。無論您身在何處，您都將始終擁有最新的檔案。

其用法於 “Nextcloud桌面客戶端手冊" 中分開介紹。

流動客戶端

在 Nextcloud 網絡介面中存取您的個人頁面，查找 Android 和 iOS 移動同步客戶端的下載連結。 或者，存取“Nextcloud 下載頁面 <https://nextcloud.com/install/>”。

很遺憾，雙向同步（即雙向同步）在 Android 客戶端尚未完全實現。請參見 此 GitHub 問題 以了解當前功能的狀態。
