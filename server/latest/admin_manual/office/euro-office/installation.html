Installation

Nextcloud Office requires a running Euro-Office Document Server that Nextcloud can reach over the network. Choose the installation method that fits your infrastructure:

Distribution

Architectures

Method

Ubuntu 24.04 LTS (Noble)

amd64, arm64

Installing on Ubuntu 24.04 LTS

Debian 12 (Bookworm)

amd64, arm64

Installing on Debian 12

Any Linux host

amd64, arm64

Installing via Docker

Minimum system requirements:

  • 4 GB RAM (8 GB recommended for multi-user deployments)

  • 10 GB disk space

  • The Document Server and Nextcloud must be able to reach each other over HTTPS in production

Note

The Document Server does not need to run on the same host as Nextcloud. A reverse proxy (nginx or Apache) in front of the Document Server is required for production deployments to expose it over HTTPS.
