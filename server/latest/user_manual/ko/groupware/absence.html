부재중 메시지 설정

휴가 혹은 병가 등의 원인으로 인하여 부재중인 경우 가용성 설정의 부재 섹션에서 부재중 메시지를 추가할 수 있습니다.

인터페이스에서 부재 기간, 짧은 메시지와 긴 메시지, 대체 사용자를 입력합니다. 이 데이터는 다음 목적으로 사용됩니다:

  1. 당신의 사용자 상태는 부재 시작 시점에 지정한 짧은 메시지로 변경되고, 종료 시점에 초기화됩니다.

  2. 그리고 당신의 개인 캘린더에 바쁨 상태로 이벤트가 생성될 것입니다. 이것으로 다른 사용자가 한가함/바쁨 기능을 사용할 때 당신이 부재중으로 확인될 수 있도록 합니다.

  3. 활성화된다면 메일 앱은 지정된 긴 메시지를 사용하여 자동 응답을 적용합니다.

  4. 토크 앱은 부재중에 다른 사용자가 1:1 대화로 당신에게 연락을 시도할 때 지정된 긴 부재중 메시지를 표시할 것이고 대체 사용자가 설정되어 있다면 대체 사용자도 표시합니다.
