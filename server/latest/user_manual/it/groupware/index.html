Groupware

Il groupware di Nextcloud è un insieme di applicazioni che consistono di un client di posta Mail (IMAP/POP3) e di un server di Calendario e Contatti (CalDAV/CardDAV) con le loro rispettive interfacce web.

Noi completiamo questi strumenti di produttività con Deck, uno strumento di gestione dei progetti che ti consente di creare tabelle di attività in stile Kanban e condividerli con la tua squadra.

Puoi trovare maggiori informazioni sul groupware di Nextcloud sul nostro sito.
