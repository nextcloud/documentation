Συγχρονισμός με Windows 10

Ημερολόγιο

  1. Στο πρόγραμμα περιήγησής σας, πλοηγηθείτε στην εφαρμογή Ημερολόγιο του Nextcloud. Κάτω από τις «Ρυθμίσεις ημερολογίου», αντιγράψτε τη διεύθυνση χρησιμοποιώντας το «Αντιγραφή διεύθυνσης CalDAV iOS/macOS» στο πρόχειρό σας.

  2. Εκκινήστε την εφαρμογή Ημερολόγιο του Windows 10. Έπειτα, κάντε κλικ στο εικονίδιο ρυθμίσεων (εικονίδιο γραναζιού) και επιλέξτε «Διαχείριση λογαριασμών».

  3. Κάντε κλικ στο «Προσθήκη λογαριασμού» και επιλέξτε «iCloud».

  4. Εισάγετε ένα email, όνομα χρήστη και κωδικό πρόσβασης. Κανένα από αυτά τα στοιχεία δεν χρειάζεται να είναι έγκυρο - όλα θα αλλάξουν στα επερχόμενα βήματα.

  5. Κάντε κλικ στο «Ολοκλήρωση». Θα πρέπει να εμφανιστεί ένα μήνυμα που υποδεικνύει ότι οι ρυθμίσεις αποθηκεύτηκαν με επιτυχία.

  6. Στο μενού «Διαχείριση Λογαριασμών», κάντε κλικ στον λογαριασμό iCloud που δημιουργήθηκε στα προηγούμενα βήματα και επιλέξτε «Αλλαγή ρυθμίσεων». Έπειτα, κάντε κλικ στο «Αλλαγή ρυθμίσεων συγχρονισμού γραμματοκιβωτίου».

  7. Κυλήστε προς το κάτω μέρος του πλαισίου διαλόγου, επιλέξτε «Προχωρημένες ρυθμίσεις γραμματοκιβωτίου». Κυλήστε ξανά προς το κάτω μέρος του πλαισίου διαλόγου και επικολλήστε τη διεύθυνση URL CalDAV σας στο πεδίο με την επιγραφή «Διακομιστής ημερολογίου (CalDAV)».

  8. Κάντε κλικ στο «Ολοκλήρωση». Εισάγετε το όνομα χρήστη και τον κωδικό πρόσβασης Nextcloud σας στα αντίστοιχα πεδία και αλλάξτε το όνομα λογαριασμού σε ό,τι προτιμάτε (π.χ. «Nextcloud Ημερολόγιο»). Κάντε κλικ στο «Αποθήκευση».

Επαφές

  1. Στην κάτω αριστερή πλευρά της Προβολής Επαφών (στο Nextcloud Επαφές) αναζητήστε ένα μικρό σύμβολο σύνδεσμου (αλυσίδας) που μοιάζει με αυτό:

που θα εμφανίσει μια διεύθυνση URL που μοιάζει κάπως έτσι: https://cloud.nextcloud.com/remote.php/dav/addressbooks/users/daniel/Thunderbird/

  1. Εκκινήστε την εφαρμογή Ημερολόγιο του Windows 10. Έπειτα, κάντε κλικ στο εικονίδιο ρυθμίσεων (εικονίδιο γραναζιού) και επιλέξτε «Διαχείριση λογαριασμών».

  2. Κάντε κλικ στο «Προσθήκη λογαριασμού» και επιλέξτε «iCloud».

  3. Εισάγετε ένα email, όνομα χρήστη και κωδικό πρόσβασης. Κανένα από αυτά τα στοιχεία δεν χρειάζεται να είναι έγκυρο - όλα θα αλλάξουν στα επερχόμενα βήματα.

  4. Κάντε κλικ στο «Σύνδεση» και έπειτα στο «Ολοκλήρωση». Θα πρέπει να εμφανιστεί ένα μήνυμα που υποδεικνύει ότι οι ρυθμίσεις αποθηκεύτηκαν με επιτυχία.

  5. Στο μενού «Διαχείριση Λογαριασμών», κάντε κλικ στον λογαριασμό iCloud που δημιουργήθηκε στα προηγούμενα βήματα και επιλέξτε «Αλλαγή ρυθμίσεων». Έπειτα, κάντε κλικ στο «Αλλαγή ρυθμίσεων συγχρονισμού γραμματοκιβωτίου».

  6. Κυλήστε προς το κάτω μέρος του πλαισίου διαλόγου, επιλέξτε «Προχωρημένες ρυθμίσεις γραμματοκιβωτίου». Κυλήστε ξανά προς το κάτω μέρος του πλαισίου διαλόγου και επικολλήστε τη διεύθυνση URL CardDAV σας στο πεδίο με την επιγραφή «Διακομιστής επαφών (CardDAV)».

  7. Κάντε κλικ στο «Ολοκλήρωση». Εισάγετε το όνομα χρήστη και τον κωδικό πρόσβασης Nextcloud σας στα αντίστοιχα πεδία και αλλάξτε το όνομα λογαριασμού σε ό,τι προτιμάτε (π.χ. «Nextcloud Επαφές»). Κάντε κλικ στο «Αποθήκευση».

Αντιμετώπιση Προβλημάτων

Αφού ακολουθήσετε όλα αυτά τα βήματα, το Nextcloud ημερολόγιό σας θα πρέπει να συγχρονιστεί. Εάν όχι, ελέγξτε το όνομα χρήστη και τον κωδικό πρόσβασής σας. Διαφορετικά, επαναλάβετε αυτά τα βήματα.

ΣΗΜΕΙΩΣΗ: Δεν θα μπορείτε να συγχρονίσετε το ημερολόγιό σας εάν έχετε ενεργοποιημένο τον έλεγχο ταυτότητας δύο παραγόντων. Ακολουθήστε τα παρακάτω βήματα για να λάβετε έναν κωδικό πρόσβασης εφαρμογής που μπορεί να χρησιμοποιηθεί με την εφαρμογή πελάτη Ημερολογίου:

  1. Συνδεθείτε στο Nextcloud. Κάντε κλικ στο εικονίδιο χρήστη σας και έπειτα κάντε κλικ στις «Ρυθμίσεις».

  2. Κάντε κλικ στην «Ασφάλεια», έπειτα εντοπίστε ένα κουμπί με την επιγραφή «Δημιουργία νέου κωδικού εφαρμογής». Δίπλα σε αυτό το κουμπί, εισάγετε «Εφαρμογή Ημερολογίου Windows 10». Έπειτα, κάντε κλικ στο κουμπί, αντιγράψτε και επικολλήστε τον κωδικό πρόσβασης. Χρησιμοποιήστε αυτόν τον κωδικό πρόσβασης αντί για τον κωδικό πρόσβασης Nextcloud σας για το Βήμα 8.

Επαφές

  1. Επαναλάβετε τα βήματα 1–7 από τις οδηγίες Ημερολογίου. Εάν έχετε ήδη ρυθμίσει τον συγχρονισμό Ημερολογίου, μπορείτε να χρησιμοποιήσετε τον ίδιο λογαριασμό για αυτό.

  2. Στις «Προχωρημένες ρυθμίσεις γραμματοκιβωτίου» επικολλήστε τη διεύθυνση URL CalDAV σας στο πεδίο με την επιγραφή «Διακομιστής επαφών (CardDAV)».

  3. Αντικαταστήστε τη διαδρομή «principals» μέσα στη διεύθυνση URL με «addressbooks».

  4. Κάντε κλικ στο «Ολοκλήρωση». Εισάγετε το όνομα χρήστη και τον κωδικό πρόσβασης Nextcloud σας στα αντίστοιχα πεδία και αλλάξτε το όνομα λογαριασμού σε ό,τι προτιμάτε (π.χ. «Nextcloud»). Κάντε κλικ στο «Αποθήκευση».
