Anhänge

Dateien in einem Chat teilen

Es gibt drei Möglichkeiten, Dateien in einem Chat zu teilen.

Erstens können Sie die Dateien einfach per Drag’n’Drop in den Chat ziehen.

Zweitens können Sie eine Datei aus Ihren Nextcloud-Dateien oder einem Dateimanager auswählen, indem Sie auf die kleine Büroklammer klicken und auswählen, aus welchem Ordner Sie die Datei holen möchten.

Sie können weitere Dateien hinzufügen, bis Sie fertig sind und sich entscheiden, die Dateien zu teilen. Sie können Ihren geteilten Dateien auch eine Textbeschriftung hinzufügen, die eine kurze Beschreibung oder einen Kontext enthält.

Alle Benutzer können auf die Dateien klicken, um sie anzusehen, zu bearbeiten oder herunterzuladen, unabhängig davon, ob sie ein Benutzerkonto haben. Für Benutzer mit einem Konto wird die Datei automatisch freigegeben, während sie für externe Gastnutzer als öffentlicher Link freigegeben wird.

Umfragen im Chat

Sie können in Gruppenchats über die zusätzlichen Aktionen für neue Nachrichten eine Umfrage erstellen.

Eine Umfrage hat zwei Einstellungen:

  • Anonyme Umfragen: Die Teilnehmer können nicht sehen, wer für welche Option gestimmt hat.

  • Mehrere Auswahlmöglichkeiten zulassen: Teilnehmer können mehr als eine Option auswählen.

Sie können Umfragen auch importieren, um sie automatisch auszufüllen, und Umfragen als JSON-Dateien exportieren, um sie lokal zu speichern.

Sie können eine Umfrage über das Umfragedialogfeld schließen.

Als Moderation können Sie die Umfrage direkt erstellen oder sie als Entwurf speichern, um sie später zu bearbeiten.

Sie finden die Entwürfe der Umfragen auf der Registerkarte Gemeinsame Elemente oder neben dem Eingabefeld für den Umfragetitel.
