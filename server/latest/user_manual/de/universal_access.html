Universeller Zugriff

Universal access is very important to us. We follow web standards and ensure everything is usable with a keyboard and assistive software such as screen readers. We aim to be compliant with the Web Content Accessibility Guidelines 2.1 on AA level, with the high contrast theme even on AAA level. We also follow the German BITV 2.0 guidelines.

Wenn Sie Fehler finden, melden Sie diese gern auf unserer Fehlerreport-Seite .Und wenn Sie mithelfen möchten, verstärken Sie unser Design-Team!

Zoom und Reaktivität

Die Nextcloud-Oberfläche ist vollständig responsiv und auf jeder Bildschirmgröße nutzbar. Sie können hinein- und hinauszoomen, um die Text- und Elementgröße nach Ihren Wünschen anzupassen. Die Navigation und die Seitenleiste können erweitert oder eingeklappt werden.

Navigieren via Tastatur

You can navigate the web interface with a keyboard, just as you would with a mouse:

  • Tab and Shift+Tab to move between elements

  • Enter or Space to activate or open the element (depending on the type of element)

  • Escape to be used to close modals, popover menus, and file viewers

  • Left arrow and Right arrow to navigate between photos in the viewer

  • Ctrl+F to focus the search field

  • Ctrl+S to save changes in editors like Nextcloud Text

For quicker navigation, we offer two „skip links“ at the beginning of the document which allow you to:

  • Wechslen Sie zu Hauptinhalt

  • Wechsle zu App-Navigation

Nextcloud Talk hat Tastaturkürzel, welche auch innerhalb der App-Einstellungen selbst dokumentiert sind:

  • C to focus the message input field

  • Escape to unfocus the message input field to be able to use shortcuts

  • F to fullscreen the chat or call

  • Während eines Anrufs:

    • M to toggle the microphone on and off

    • V to toggle video on and off

    • Space for push to talk or push to mute

    • R to raise or lower hand

Nextcloud Mail verfügt ebenfalls über Kurzbefehle, die auch in den Einstellungen der App selbst dokumentiert sind:

  • C to compose a new message

  • Left arrow to switch to a newer message

  • Right arrow to switch to an older message

  • S to toggle a message as favorite

  • U to toggle a message unread

  • Del to delete a message

  • Ctrl+Enter to send

  • R to refresh and load new mails

Inbegriffene Designs

Wir bieten einige Designs an, die Sie zur besseren Barrierefreiheit aktivieren können:

  • Hochkontrast-Design: Ein Hochkontrast-Modus zur einfacheren Navigation. Die visuelle Qualität wird reduziert, doch die Lesbarkeit wird erhöht.

  • Dunkles Design: Ein dunkles Design zur Schonung Ihrer Augen, das Beleuchtung und Helligkeit insgesamt reduziert. Es befindet sich noch immer in Entwicklung, also bitte melden Sie Fehler, die Sie finden.

  • Legasthenie-Schriftart: OpenDyslexic ist eine freie Schriftart mit dem Ziel, einige häufige Lesefehler zu vermeiden, die durch Legasthenie verursacht werden.

Um die Einstellungen für die Barrierefreiheit aufzurufen:

  1. Öffnen Sie das Einstellungsmenü am Ende des Kopfbereichs

  2. Settings auswählen

  3. In der Navigation Barrierefreiheit* auswählen

Bemerkung

Die Kontraste von Elementen können bei benutzerdefinierten Designs variieren. Zum Beispiel wird die primäre Design-Farbe als Hintergrundfarbe für Kopfbereich, Anmeldeseite und primäre Schaltflächen benutzt. Falls dies Probleme mit dem Kontrast verursacht, bitten Sie Ihre Adminstration um Hilfe.
