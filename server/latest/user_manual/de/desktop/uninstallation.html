Deinstallation

Sie können den Files Desktop Client sicher mit den in Ihrem Betriebssystem integrierten Tools deinstallieren.

Allgemeine Anweisungen

In jedem Betriebssystem:

  1. Sicherstellen, dass Sie den Desktop-Client beenden, bevor dieser entfernt wird.

  2. Deinstallieren Sie den Desktop-Client.

  3. Sie sollten außerdem berücksichtigen, dass durch die Deinstallation des Desktop-Clients die Konfigurationsdatei des Benutzers <https://docs.nextcloud.com/server/latest/admin_manual/desktop/configfile.html#configuration-file>`_ und die synchronisierten Daten nicht entfernt werden.

    Zum Löschen von Daten (Synchronisierungsordner) empfiehlt sich die Serverfunktion Remote-Löschen <siehe <https://nextcloud.com/de/blog/nextcloud-desktop-client-2-6-1-brings-remote-wipe-dark-mode-support-to-mac-os-x-and-more/>`_. Diese Funktion ist auf allen Clients verfügbar.

Nachfolgend finden Sie plattformspezifische Anweisungen.

Windows

Es gibt zwei Möglichkeiten, den Desktop-Client zu entfernen:

  1. Über die Funktion Programme hinzufügen oder entfernen.

  2. Sie können die Befehlszeilenoptionen von msiexec verwenden:

msiexec /uninstall Nextcloud-x.y.z-x64.msi /quiet``

  1. Die Benutzerkonfiguration <https://docs.nextcloud.com/server/latest/admin_manual/desktop/configfile.html#configuration-file> befindet sich unter %APPDATA%\Nextcloud\nextcloud.cfg.

macOS

Neben dem Löschen des Desktop-Clients aus dem Ordner Programme müssen Sie möglicherweise auch alle zugehörigen Daten entfernen, insbesondere wenn Sie virtuelle Dateien verwenden.

  1. Um die Software einfach zu deinstallieren, können Sie dies über das Launchpad oder den Finder tun <https://support.apple.com/en-us/102610>`_.

  2. Zur vollständigen Löschung aller zugehörigen Daten können folgende Befehle verwendet werden:

rm -rf "$HOME/Library/Application Scripts/com.nextcloud.desktopclient"*
rm -f  "$HOME/Library/Application Support/CrashReporter/Nextcloud_"*
rm -rf "$HOME/Library/Application Support/Nextcloud"
rm -rf "$HOME/Library/Caches/Nextcloud"
rm -rf "$HOME/Library/Containers/com.nextcloud.desktopclient"*
rm -rf "$HOME/Library/Group Containers/NKUJUXUJ3B.com.nextcloud.desktopclient"
rm -rf "$HOME/Library/Group Containers/com.nextcloud.desktopclient"
rm -f  "$HOME/Library/LaunchAgents/com.nextcloud.desktopclient.plist"
rm -rf "$HOME/Library/Preferences/Nextcloud"
rm -f  "$HOME/Library/Preferences/com.nextcloud.desktopclient.plist"

  1. Ab Version 33.0.0 befindet sich die Benutzerkonfigurationsdatei unter $HOME/Library/Containers/com.nextcloud.desktopclient/Data/Library/Preferences/Nextcloud/nextcloud.cfg. In älteren Versionen befindet sie sich unter $HOME/Library/Preferences/Nextcloud/nextcloud.cfg.

Linux

Das hängt davon ab, wie Sie den Desktop-Client installiert haben:

  1. Wenn Sie die Nextcloud AppImage-Datei verwenden, können Sie die AppImage-Datei einfach löschen.

  2. Wenn Sie den Desktop-Client mit Ihrem Paketmanager installiert haben, können Sie ihn auch damit deinstallieren. Unter Ubuntu können Sie beispielsweise folgenden Befehl verwenden:

sudo apt remove nextcloud-desktop

  1. Die Benutzerkonfiguration befindet sich unter $HOME/.config/Nextcloud/nextcloud.cfg.
