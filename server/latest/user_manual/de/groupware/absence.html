Abwesenheitsnachricht setzen

Sollten Sie wegen Urlaub, Krankheit oder Ähnlichem abwesend sein, können Sie in den Verfügbarkeit-Einstellungen im Bereich Abwesenheit eine Abwesenheitsnachricht hinzufügen.

Die Schnittstelle fragt den Zeitpunkt der Abwesenheit, eine kurze und eine lange Nachricht sowie optional einen Ersatzbenutzer ab. Diese Daten werden für folgende Zwecke verwendet:

  1. Ihr Benutzerstatus ändert sich mit Beginn Ihrer Abwesenheit auf die Kurznachricht und wird nach deren Ende zurückgesetzt.

  2. An event with status busy will be created in your personal calendar. This allows others to see that you are not available when they use the free/busy feature.

  3. Wenn aktiviert, wendet die Mail-App einen Autoresponder mit der langen Nachricht an.

  4. Die Talk-App zeigt die lange Abwesenheitsnachricht anderen an, wenn diese versuchen, Sie während Ihrer Abwesenheit in einem 1:1-Chat zu erreichen, sowie dem Ersatzbenutzer, falls festgelegt.
