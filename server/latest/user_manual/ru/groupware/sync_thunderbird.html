Синхронизация с Thunderbird

Thunderbird — это многофункциональный и зрелый почтовый клиент, который можно превратить в полноценный менеджер личной информации (PIM). Начиная с версии 102, он поддерживает синхронизацию адресных книг через CardDAV и автоматическое обнаружение календарей и адресных книг, доступных на сервере.

Рекомендуемый метод

Начиная с Thunderbird 102, протоколы CardDAV & CalDAV поддерживаются нативно.

Контакты

  1. В представлении адресной книги нажмите стрелку вниз рядом с надписью Новая адресная книга и выберите Добавить адресную книгу CardDAV.

  2. В следующем окне введите свое Имя пользователя и Местоположение (URL-адрес сервера).

  3. В следующем окне будет запрошено ваше имя пользователя и пароль для этой учётной записи.

  4. Предыдущее окно будет обновлено и предложит вам, какие адресные книги вы хотите синхронизировать.

  5. Выберите и нажмите Продолжить.

Если позже вы захотите добавить новую адресную книгу, вы можете повторить все эти шаги, и будут предложены только те книги, которые ещё не синхронизированы.

Примечание

Если ваша учетная запись использует двухфакторную аутентификацию (2FA), вам понадобится выделенный пароль приложения для входа в систему, а не обычный пароль.

Календари

  1. Перейдите в представление календаря в Thunderbird и нажмите кнопку Новый календарь… в нижней части левой боковой панели.

  2. Выберите В сети

  3. Введите Имя пользователя и Расположение (URL-адрес сервера), затем нажмите Найти календари.

  4. Выберите календари, которые вы хотите добавить, и нажмите Подписаться

То же самое здесь, если вы позже захотите добавить больше календарей, просто повторите процедуру.

Альтернатива: использование надстройки CardBook (только для контактов)

CardBook это продвинутая альтернатива адресной книге Thunderbird, поддерживающая CardDAV.

  1. Нажмите на иконку Cardbook в верхнем правом углу Thunderbird:

  2. В Cardbook:

    • Адресная книга > Новая адресная книга Удалённая > Далее

    • Выберите CardDAV, введите адрес вашего сервера Nextcloud, ваше имя пользователя и пароль

  3. Нажмите «Подтвердить», нажмите «Далее», затем выберите имя адресной книги и снова нажмите «Далее».

  4. Когда вы закончите, CardBook синхронизирует ваши адресные книги. Вы всегда можете запустить синхронизацию вручную, нажав «Синхронизировать» в верхнем левом углу CardBook:
