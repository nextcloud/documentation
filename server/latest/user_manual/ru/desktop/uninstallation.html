Удаление

Файлы настольного клиента можно безопасно удалить с помощью встроенных инструментов вашей операционной системы.

Общие инструкции

В каждой операционной системе:

  1. Обязательно убедитесь что вышли из настольного клиента перед его удалением.

  2. Удалить настольный клиент.

  3. You may also need to take into consideration that uninstalling the desktop client will not remove the user’s configuration file and synced data.

    Для удаления данных (папок синхронизации) рассмотрите возможность использования функции сервера удалённая очистка. Эта функция доступна для всех клиентов.

Ниже приведены инструкции для конкретной платформы.

Windows

Существует два способа удаления настольного клиента:

  1. Использование Установка и удаление программ.

  2. Вы можете использовать параметры командной строки msiexec:

msiexec /uninstall Nextcloud-x.y.z-x64.msi /quiet``

  1. The user’s configuration is located at %APPDATA%\Nextcloud\nextcloud.cfg.

macOS

Помимо удаления настольного клиента из папки Приложения, вам также может потребоваться удалить все связанные данные, особенно если вы используете виртуальные файлы.

  1. Чтобы просто удалить программное обеспечение: вы можете сделать это с Launchpad или с помощью Finder.

  2. Для полного удаления всех связанных данных можно использовать следующие команды:

rm -rf "$HOME/Library/Application Scripts/com.nextcloud.desktopclient"*
rm -f  "$HOME/Library/Application Support/CrashReporter/Nextcloud_"*
rm -rf "$HOME/Library/Application Support/Nextcloud"
rm -rf "$HOME/Library/Caches/Nextcloud"
rm -rf "$HOME/Library/Containers/com.nextcloud.desktopclient"*
rm -rf "$HOME/Library/Group Containers/NKUJUXUJ3B.com.nextcloud.desktopclient"
rm -rf "$HOME/Library/Group Containers/com.nextcloud.desktopclient"
rm -f  "$HOME/Library/LaunchAgents/com.nextcloud.desktopclient.plist"
rm -rf "$HOME/Library/Preferences/Nextcloud"
rm -f  "$HOME/Library/Preferences/com.nextcloud.desktopclient.plist"

  1. From version 33.0.0 the user’s configuration is located at $HOME/Library/Containers/com.nextcloud.desktopclient/Data/Library/Preferences/Nextcloud/nextcloud.cfg. On older versions it is located at $HOME/Library/Preferences/Nextcloud/nextcloud.cfg.

Linux

Это зависит от того, как вы установили настольный клиент:

  1. Если вы используете Nextcloud AppImage, вы можете просто удалить файл AppImage.

  2. Если вы использовали менеджер пакетов для установки клиента для настольных компьютеров, вы также можете использовать его для удаления клиента для настольных компьютеров. Например, в Ubuntu вы можете использовать следующую команду:

sudo apt remove nextcloud-desktop

  1. The user’s configuration is located at $HOME/.config/Nextcloud/nextcloud.cfg.
