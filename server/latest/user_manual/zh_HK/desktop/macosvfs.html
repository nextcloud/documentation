macOS 虛擬檔案客戶端

在 macOS 上，客戶端可透過檔案提供者延伸功能，將 Nextcloud 檔案無縫整合至系統。所有新設定的 Nextcloud 帳戶預設均會啟用此整合。

支持的功能

  • 讓檔案或整個資料夾可離線使用

  • 透過移除本機副本（不刪除項目）釋放磁碟空間

  • 智能及自動移除本機資料副本

  • 即使未下載檔案，亦可在 Finder 中預覽

  • 支援 Apple 專用格式，例如 Pages、Numbers 及 Keynote 檔案

  • 支援伺服器端檔案鎖定（視乎伺服器是否支援）

  • 支持"近端編輯"

  • 與其他用戶分享

  • 伺服器端操作直接整合於 Finder 右鍵選單

  • 自動偵測伺服器端變更

配置

虛擬檔案相關設定可於 Nextcloud 桌面客戶端的設定視窗中，按帳戶個別調整。

可在此啟用或停用與 Finder 的整合。

若在仍有未同步變更時停用檔案提供者延伸功能，macOS 會將未同步的項目儲存至一個資料夾，並在停用後自動顯示該資料夾。

Finder 集成

在 macOS 上，Nextcloud 等遠端儲存會於 Finder 側邊欄顯示為獨立位置。實際檔案儲存位置則由 macOS 決定。

Note

為加快偵測伺服器端變更，建議在你的 Nextcloud 伺服器上啟用 notify_push 應用程式。此應用程式會在變更發生時即時通知桌面客戶端，從而縮短變更在 Finder 中顯示所需的時間。否則，客戶端需定期輪詢伺服器，導致伺服器變更與本機顯示之間出現較大延遲。

同步狀態指示器

與傳統同步資料夾類似，Finder 會在項目旁顯示狀態指示。不同於傳統同步資料夾中的自訂圖示，這些標準化指示由 macOS 提供，以確保用戶在系統上使用不同雲端儲存應用程式時，均有一致的顯示效果。

  • 向下箭咀雲圖示：該項目及其所有子項目尚未下載。在有網絡連線時可下載。

  • 空心雲圖示：該項目在目前的本機狀態下尚未完全上傳。

  • 帶刪除線的雲圖示：該項目已排除於同步之外。

  • 圓餅圖圖示：該項目正在上載或下載，並顯示進度。

  • Filled circle with a pin: The item is available offline and will be kept locally.

  • 沒有圖示：該項目可離線使用，且為最新狀態。

選項單操作

檔案提供者延伸功能亦會透過 Finder 右鍵選單提供 Nextcloud 專屬功能。

Keep Downloaded

Files and folders can be marked to be kept downloaded and available offline permanently. If this is chosen on folders, it will also apply to all their contents. This is especially useful for users with limited or no network access, as it ensures that they can always access their important files without needing to worry about connectivity. macOS will not free up local disk space by evicting items which are marked to be kept downloaded, even if they have not been accessed for a long time.

This can be undone by selecting "Allow automatic freeing up space" on the same items.

To always keep everything in an account available locally, you can select "Always keep downloaded" on the location root in the Finder sidebar.

鎖定

如伺服器支援檔案鎖定，客戶端會在 Finder 中提供手動鎖定及解除鎖定檔案的功能。

分享

如伺服器支援分享且該項目允許分享，你可直接於 Finder 的右鍵選單建立新的分享或管理現有分享，方式與 Nextcloud 網頁介面相同。

檔案操作

如伺服器安裝了為所選檔案類型提供檔案操作的應用程式，這些操作亦會顯示於 Finder 的右鍵選單中。這讓你可直接在 Finder 使用 Nextcloud 實例的伺服器端功能。

已知問題

macOS 延伸功能衝突

由於 Apple 在 macOS 上的技術限制，無法同時啟用傳統同步資料夾的 Finder 整合與虛擬檔案整合。因此，在啟用檔案提供者延伸功能時，傳統同步資料夾將不會顯示狀態圖示或提供右鍵選單選項。

別名檔案

當首次在尚未下載該檔案的裝置上開啟儲存在 Nextcloud 的 macOS 別名檔案時，該檔案可能會被誤當作二進位文件於文字編輯器中開啟，而非導向其目標位置。這是因為 macOS 會在下載前決定檔案的開啟方式，而別名檔案在伺服器上並無可識別的副檔名或類型資訊，唯一識別方法是讀取其內容。當檔案被開啟或下載一次後，Nextcloud Desktop 會識別其為別名並在本機記錄相關資訊，其後開啟將會正常運作。為完全避免此問題，請在 Finder 中右鍵點擊該別名檔案，並於首次開啟前選擇 Keep Downloaded。
