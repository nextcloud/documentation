使用端對端加密

如伺服器端已啟用，Nextcloud 可讓你在裝置上對檔案進行加密。此功能稱為端對端加密（E2EE），因為檔案只會在你的裝置上加密及解密。伺服器不會接觸未加密的檔案，進一步保障用戶私隱及資料安全。

啟用 E2EE

如系統管理員已啟用 End-to-End Encryption app，你可在其中一部裝置上開始使用。請開啟 Nextcloud 客戶端的設定，並尋找 E2EE 加密設定。

Android
iOS
Web

Warning

無法在瀏覽器中為資料夾啟用加密。此操作必須透過客戶端應用程式完成，包括桌面或流動客戶端。

新增 E2EE 裝置

Desktop
Android
iOS
Web

在瀏覽器中，需先於個人設定啟用 E2EE。這是因為 E2EE 在瀏覽器環境中的安全性較低，需完全信任系統管理員不會修改瀏覽器所執行的原始碼。目前 E2EE 資料夾僅支援唯讀，因此無法在瀏覽器中新增、刪除、編輯或分享 E2EE 檔案。


顯示助記詞

助記詞是一組用於加密及解密檔案的單詞列表。請務必妥善保管此助記詞，因為若你失去裝置存取權，它將是唯一可用來存取檔案的方法。如遺失助記詞，將無法再存取你的檔案。

Warning
