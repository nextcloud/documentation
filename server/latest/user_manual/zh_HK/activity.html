使用「動態」應用程式

「動態」應用程式會追蹤 Nextcloud 內發生的所有事情。它會記錄檔案變更、分享、日曆更新等事件，讓你按時間順序概覽在何時發生了甚麼事。

「動態」應用程式預設為啟用。

查看你的動態串流

若要查看你的動態串流，請點選頂部導覽列的**動態**圖示，或在瀏覽器中前往 /apps/activity。

串流會按日期分組顯示所有事件。每個條目都會顯示發生的內容、受影響的檔案或物件，以及事件發生的時間。如果你安裝了日曆、聯絡人或 Talk 等其他應用程式，相關條目也會出現在串流中。

篩選動態

左側側邊欄提供篩選器以縮小動態串流的範圍：

  • **所有動態**會顯示所有內容。

  • **由你執行**僅顯示由你自己觸發的動態。

  • **由他人執行**僅顯示由其他用戶觸發的動態。

  • **檔案變更**僅顯示檔案與資料夾事件（建立、修改、刪除、重新命名、移動）。

視乎你的 Nextcloud 實例中安裝了哪些應用程式，可能會出現額外的篩選器（例如：收藏項目、日曆、聯絡人）。

檔案側邊欄中的動態

你亦可以直接在「檔案」應用程式中查看特定檔案或資料夾的動態。選擇檔案，打開側邊欄並點選**動態**分頁。這將僅顯示與該特定檔案相關的事件，例如其建立、修改、分享或標記的時間。

RSS 摘要

「動態」應用程式可以將你的動態串流以 RSS 摘要形式提供，讓你透過任何摘要閱讀器關注 Nextcloud 動態。

若要啟用 RSS 摘要：

  1. 開啟「動態」應用程式。

  2. 點選左側側邊欄底部的**動態設定**。

  3. 開啟**啟用 RSS 摘要**開關。

  4. 複製出現的 RSS 摘要連結。

Note

RSS 摘要連結包含一個金鑰。請勿將其與他人分享，因為它提供對你動態串流的未經授權訪問。

通知設定

你可以選擇希望如何收到不同類型動態的通知。前往**設定** > 個人 > **通知**來設定你的偏好。

設定按類別組織（例如：檔案、分享、日曆、聯絡人與任務）。對於每種動態類型，你可以選擇接收：

  • **郵件**通知（電子郵件）

  • **推送**通知（手機及桌面）

勾選或取消勾選對應的方塊，以啟用或停用每種動態類型的通知。

電子郵件通知頻率

在通知矩陣下方，你可以選擇電子郵件通知的發送頻率：

  • **盡快**在每個事件發生後不久發送電子郵件。

  • **每小時**批次處理通知並每小時發送一次。

  • **每日**每天發送一封電子郵件。

  • **每週**每週發送一封電子郵件。

每日動態摘要

你可以啟用每日摘要電子郵件，每天早上發送前一日的動態概覽。此選項位於通知頻率設定下方的一個獨立開關：

  • 勾選**在早上發送每日動態摘要**以啟用它。

每日摘要獨立於上述的逐項事件通知，並提供所有動態的便捷摘要。
