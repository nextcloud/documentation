傳送訊息

使用 Markdown

你可以使用 Markdown 語法強化訊息內容。支援的元素包括：

標題與分隔線

# Heading 1
## Heading 2
### Heading 3
#### Heading 4
##### Heading 5
###### Heading 6

Heading
===
Normal text
***
Normal text

行內格式

**bold text** __bold text__
*italicized text* _italicized text_
`inline code` ``inline code``
```
.code-block {
  display: pre;
}
```

清單

1. Ordered list
2. Ordered list

* Unordered list
- Unordered list
+ Unordered list

引用

> blockquote
second line of blockquote

任務清單

- [ ] task to be done
- [x] completed task

表格

Column A | Column B
-- | --
Data A | Data B

插入表情符號

你可以使用輸入欄左側的選擇器加入表情符號。

Smart Picker

Smart Picker 讓你更輕鬆在對話中插入連結、檔案或其他內容。只需選擇要插入的內容類型（files、Talk 對話、Deck 卡片、GIF 等）。你亦可以在聊天輸入欄輸入 / 開啟選擇器。

多種整合應用程式可擴展 Smart Picker，提供額外的內容類型，例如 GitHub 和 GitLab issues、Giphy GIF 等。請向系統管理員查詢你的實例可用的整合功能。

回覆訊息及其他功能

將滑鼠移至訊息上時會出現箭頭，點擊即可回覆該訊息。

在 ... 選單中，你亦可以選擇私下回覆，系統會開啟一對一對話。

你亦可以在此建立訊息的直接連結，或將其標記為未讀，以便下次進入聊天時返回該位置。如為檔案，則可於 Files 中檢視。

靜音訊息

如果你不希望在深夜打擾他人，可以使用聊天靜音模式。啟用後，其他參與者將不會收到你的訊息通知。

排程傳送訊息

如果你希望訊息於指定時間而非即時傳送，可以使用排程功能。只需在輸入欄旁的快速操作中選擇日期及時間即可。

你可以點擊輸入欄旁的時鐘圖示查看所有已排程的訊息，並可進行編輯、重新排程或刪除。

聊天摘要

當啟用 AI 助手後，如對話中有超過 100 條未讀訊息，系統可生成摘要。你可以按下聊天中第一條未讀訊息上方顯示的按鈕來產生摘要。
