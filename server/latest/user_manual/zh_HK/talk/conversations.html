對話

聊天同通話都係喺「對話」入面進行。 你可以建立任意數量嘅對話。對話主要分為以下幾類：

1. 私人（一對一）對話

喺呢度你可以同另一位 Talk 用戶進行私人聊天或通話。

喺內容側邊欄，你可以搵到對方嘅附加資訊，例如電郵地址、電話號碼，或者佢喺個人檔案入面分享嘅其他詳情。

除咗你同對方之外，無人可以睇到呢個對話或加入通話。 你可以透過加入更多人，將進行中嘅通話擴展到一個新嘅群組對話。 通話會喺嗰度繼續，唔會中斷。

如果用戶唔得閒，並喺「個人設定 > 可用性」入面設定咗**辦公室外（休假）**狀態，你就會喺對話入面睇到額外資訊，例如提供嘅說明、缺勤日期或其職務代理人。

2. 群組對話

群組對話可以容納任意數量嘅成員。 你可以在建立對話時，或者喺現有對話嘅「參與者」分頁入面，加入內部用戶、郵件訪客、群組或團隊。

群組對話可以透過公開連結分享，等訪客都可以加入聊天同通話。 對話亦可以開放畀已註冊用戶（或「訪客」應用程式嘅用戶），等佢哋可以探索並加入呢個對話。

3. 給自己的筆記

呢個係你同自己嘅專屬對話。 呢度嘅訊息喺編輯或刪除方面無限制。 你可以用佢嚟：

  • 記筆記：寫低諗法、提醒或你想隨手查閱嘅重要資訊。

  • 建立待辦事項清單：使用 Markdown 語法為需要完成嘅工作建立清單。

  • 轉寄其他聊天訊息：使用訊息選單將其他對話中嘅重要訊息轉寄到「給自己的筆記」。

4. 一次性對話

呢啲對話適用於某些特殊情況，並且只會存在一段有限嘅時間。 保留期限可以由系統管理員設定：

  • 即時會議：呢啲對話係為咗快速、臨時嘅會議而設。 你可以喺 Talk 控制面板即時發起。

  • 活動對話：當喺「日曆」應用程式將其設定為活動地點時建立。

  • 電話對話：專門用於 SIP 撥入及撥出電話（需要 SIP 閘道器）。

  • 影片驗證：當有人嘗試存取受密碼及影片驗證保護嘅公開連結時建立（通話結束後會立即刪除）。

Talk 控制面板

Talk 控制面板係你管理同存取對話嘅中心樞紐。佢提供咗以下項目嘅概覽：

  • 私人聊天中未讀嘅標記同訊息；

  • 排定稍後處理嘅訊息提醒；

  • 已排程嘅會議，附有活動詳情同加入會議嘅捷徑掣；

  • 快速建立新對話、加入公開對話或檢查媒體裝置嘅捷徑操作。

建立對話

你可以透過搜尋用戶、群組或團隊嘅名稱並點擊佢嚟建立私人（一對一）聊天。如果係單一用戶，系統會立即建立對話，你可以開始聊天。 如果係群組或小圈子，你可以在建立對話同加入參與者之前，先選擇名稱同設定。

如果你想建立自定義嘅群組對話，請點擊搜尋欄同篩選按鈕旁邊嘅掣，然後點擊「建立新對話」。

之後你可以為對話命名、填寫說明、設定頭像（上載相片或使用表情符號），並選擇對話是否開放畀外部用戶，以及伺服器上嘅其他用戶是否可以睇到同加入該對話。

喺第二步，你可以加入參與者並完成對話嘅建立。

確認之後，你將會被重新導向到新對話，可以立即開始溝通。

篩選對話

你可以使用搜尋欄旁邊嘅篩選掣嚟篩選對話。 篩選選項包括：1. 未讀標記：查看未讀嘅私人對話，或者你有被提及（Tag）嘅群組對話。 2. 未讀訊息：查看你參與嘅所有對話中嘅未讀訊息。 3. 活動對話：查看所有為未來或過去活動而建立嘅對話。

你隨後可以喺篩選選單中清除篩選條件。

封存對話

你可以封存嗰啲唔需要再喺主對話清單顯示嘅對話。封存後，對話會移至「已封存對話」部分。 封存嘅對話唔會喺主清單出現，但仍然會根據其設定中嘅通知層級發送通知。

你可以透過導覽欄底部嘅按鈕存取該清單。

管理對話

你喺自己建立嘅新對話中永遠係版主（管理員）。 喺參與者清單中，你可以點擊用戶名右側嘅「...」選單，將其他參與者提升為版主、分配自定義權限，或將佢哋從對話中移除。

更改加入公開對話用戶嘅權限，亦會將佢哋永久加入該對話。

版主可以配置對話。 從頂部對話嘅「...」選單中選擇「對話設定」即可進入設定頁面。

喺呢度你可以配置說明、訪客權限、對話是否對伺服器上嘅其他人可見等設定。

封鎖參與者

為咗確保討論安全受控，版主可以封鎖對話參與者。 呢個功能適用於內部用戶同訪客；針對訪客，系統亦會封鎖佢哋嘅 IP 地址。

喺參與者清單中，選擇該用戶或訪客，然後點擊「移除參與者」。

喺嗰度，勾選「同時從此對話封鎖」核取方塊，並提供封鎖原因。 被封鎖嘅用戶會被移除，並無法再次加入。

你稍後可以喺對話設定嘅「管理」部分搵到被封鎖用戶清單。 喺呢度，你可以睇到封鎖原因，並喺需要時解除封鎖。

訊息過期

版主可以喺「對話設定」嘅「管理」部分配置訊息過期時間。 訊息一旦到達過期時間，就會自動從對話中刪除。 可選嘅期限包括：1 小時、8 小時、1 日、1 星期、4 星期，或者「永不」（預設值）。

通知與私隱

預設情況下，Nextcloud Talk 會喺以下情況通知你：

  • 私人對話中有新訊息；

  • 有人回覆你傳送嘅訊息；

  • 訊息中提及你或你所屬嘅群組/團隊；

  • 你參與嘅對話中發起了通話。

你可以喺對話設定中更改呢啲行為。 此外，你仲可以配置：

  • 重要對話：即使你處於「請勿打擾」模式，你仍然會收到新訊息通知；

  • 敏感對話：訊息內容唔會顯示喺對話清單入面，通知中嘅內容亦會被隱藏。
