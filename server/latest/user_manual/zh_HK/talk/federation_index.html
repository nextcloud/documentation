Federation

通過聯邦功能，用戶可以在不同的聯邦 Talk 實例之間創建對話，並像在同一伺服器上那樣使用 Talk 功能。

此功能必須由系統管理員啟用。

正在發送邀請

如要接收邀請，對方需要你的 CloudID。你可在「Personal settings」中的「Sharing」找到你的 CloudID，其格式類似 user@cloud.example.com。

對話的主持人可以向不同伺服器上的參與者發送邀請：

接受邀請

收到通知時，用戶會在對話列表上方看到待處理邀請的計數。

點擊後，將提供有關邀請方的更多信息，用戶可以選擇接受或拒絕邀請。

接受邀請後，該對話將像其他對話一樣出現在列表中。

你可以使用它與來自其他聯邦伺服器的參與者聊天、加入通話，以及使用其他可用的 Talk 功能。
