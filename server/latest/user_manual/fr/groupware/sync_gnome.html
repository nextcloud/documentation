Synchronisation avec le bureau GNOME

Le bureau GNOME intègre un support pour le calendrier, les contacts et les tâches de Nextcloud qui seront affichés par le gestionnaire d’informations personnelles (PIM) d’Evolution ou par les applications Calendrier, Tâches et Contacts. De même, Fichiers s’intègre au gestionnaire de fichiers Nautilus via WebDAV. Ce dernier ne fonctionne que lorsque l’ordinateur est connecté.

Cela peut être fait en suivant ces étapes :

  1. Dans les paramètres GNOME, ouvrir Comptes en ligne.

  2. Sous « Ajouter un compte » choisir Nextcloud :

  3. Saisissez l’URL de votre serveur, votre nom d’utilisateur et votre mot de passe. Si vous avez activé l’authentification à deux facteurs (2FA), vous devez générer un jeton/mot de passe, car GNOME Online Accounts ne prend pas encore en charge la connexion WebFlow de Nextcloud (En savoir plus) :

  4. Dans la fenêtre suivante, sélectionner les ressources auxquelles GNOME devrait accéder et cliquer sur la croix en haut à droite pour fermer :

Les tâches, calendriers et contacts Nextcloud devraient désormais être visibles dans Evolution PIM, ainsi que dans les applications Tâches, Contacts et Calendriers.

Les fichiers seront affichés comme une ressource WebDAV dans le gestionnaire de fichiers Nautilus et seront également disponibles dans les boîtes de dialogue d’ouverture/enregistrement de fichiers GNOME. Les documents doivent être intégrés dans l’application GNOME Documents.

Toutes les ressources devraient aussi être consultables depuis n’importe où en appuyant sur la touche Windows et en entrant une expression à rechercher.
