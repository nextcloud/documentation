Désinstallation

Il est possible de désinstaller le client de bureau en toute sécurité en utilisant les outils intégrés à votre système d’exploitation.

Instructions générales

Quel que soit le système d’exploitation :

  1. Assurez-vous de quitter le client de bureau avant de le retirer.

  2. Désinstaller le client de bureau.

  3. Prenez également en considération le fait que la désinstallation du client de bureau ne supprimera pas le fichier de configuration utilisateur ni les données synchronisées.

    Pour supprimer des données (dossiers synchronisés), utilisez la fonction d’effacement distant du serveur. Cette fonction est disponible pour tous les clients.

Vous trouverez ci-dessous les instructions spécifiques à chaque plateforme.

Windows

Il y a deux façons de retirer le client de bureau :

  1. En passant par les Applications installées.

  2. Vous pouvez utiliser les options de ligne de commande msiexec  :

msiexec /uninstall Nextcloud-x.y.z-x64.msi /quiet``

  1. Le fichier de configuration utilisateur se trouve dans %APPDATA%\Nextcloud\nextcloud.cfg.

macOS

Outre la suppression du client de bureau situé dans le dossier Applications, vous devrez peut-être également supprimer toutes les données associées, notamment en cas d’utilisation des fichiers virtuels.

  1. Pour simplement désinstaller le logiciel : vous pouvez le faire depuis le Launchpad ou le Finder.

  2. Pour supprimer complètement toutes les données associées, les commandes suivantes peuvent être utilisées :

rm -rf "$HOME/Library/Application Scripts/com.nextcloud.desktopclient"*
rm -f  "$HOME/Library/Application Support/CrashReporter/Nextcloud_"*
rm -rf "$HOME/Library/Application Support/Nextcloud"
rm -rf "$HOME/Library/Caches/Nextcloud"
rm -rf "$HOME/Library/Containers/com.nextcloud.desktopclient"*
rm -rf "$HOME/Library/Group Containers/NKUJUXUJ3B.com.nextcloud.desktopclient"
rm -rf "$HOME/Library/Group Containers/com.nextcloud.desktopclient"
rm -f  "$HOME/Library/LaunchAgents/com.nextcloud.desktopclient.plist"
rm -rf "$HOME/Library/Preferences/Nextcloud"
rm -f  "$HOME/Library/Preferences/com.nextcloud.desktopclient.plist"

  1. À partir de la version 33.0.0, le fichier de configuration utilisateur se trouve dans $HOME/Library/Containers/com.nextcloud.desktopclient/Data/Library/Preferences/Nextcloud/nextcloud.cfg. Pour les versions antérieures, il se trouve dans $HOME/Library/Preferences/Nextcloud/nextcloud.cfg.

Linux

Cela dépend de la façon dont vous avez installé le client de bureau :

  1. Si vous passez par Nextcloud AppImage, vous pouvez simplement supprimer le fichier AppImage.

  2. Si vous avez utilisé votre gestionnaire de paquets pour installer le client de bureau, vous pouvez également l’utiliser pour le désinstaller. Par exemple, sous Ubuntu, vous pouvez utiliser la commande suivante :

sudo apt remove nextcloud-desktop

  1. Le fichier de configuration utilisateur se trouve dans $HOME/.config/Nextcloud/nextcloud.cfg.
