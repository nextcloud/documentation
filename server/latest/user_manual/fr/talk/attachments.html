Pièces jointes

Partage de fichiers dans une discussion

Le partage de fichiers dans une discussion peut se faire de trois manières.

La première consiste à faire un simple glisser-déplacer des fichiers dans la discussion.

La deuxième consiste à récupérer le fichier dans vos fichiers Nextcloud ou dans votre gestionnaire de fichiers. Pour cela, cliquez sur le petit trombone , puis sélectionnez l’emplacement du fichier à partager.

Vous pouvez ensuite ajouter d’autres fichiers à partager. Vous pouvez également ajouter une légende sous forme de texte à chaque fichier que vous partagez, pour en fournir une petite description ou en expliquer le contexte.

All users will be able to click the files to view, edit or download them, regardless of whether they have a Nextcloud account. Users with an account will have the file automatically shared with them while external guest users will get them shared as a public link.

Sondages dans les discussions

Vous pouvez créer un sondage dans les discussions de groupe via les nouvelles actions supplémentaires sur les messages.

Chaque sondage comporte deux paramètres :

  • Sondage anonyme : les participants ne peuvent pas voir qui a voté pour quel choix.

  • Réponses multiples : les participants peuvent sélectionner plusieurs choix.

Vous pouvez également importer des sondages pour les remplir automatiquement et exporter les sondages sous forme de fichiers JSON et les enregistrer en local.

Vous pouvez fermer un sondage à partir de la boîte de dialogue du sondage.

En tant que modérateur, vous pouvez directement créer le sondage ou bien l’enregistrer en brouillon pour le modifier plus tard.

Vous pouvez retrouver les brouillons de sondages dans l’onglet Éléments partagés ou à côté du champ de saisie du titre du sondage.
