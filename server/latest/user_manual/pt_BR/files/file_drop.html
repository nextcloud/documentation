Fazer uploads anônimos

Você pode criar seus próprios diretórios de upload especiais para que outras pessoas possam fazer upload de arquivos para você sem ter que fazer login no servidor e sem ser um usuário do Nextcloud. Eles não terão permissão para ver o conteúdo deste diretório ou fazer quaisquer alterações. Esta é uma excelente alternativa para enviar grandes anexos por e-mail, usando um servidor FTP ou usando serviços de compartilhamento de arquivos comerciais.

Configurar seu próprio depósito de arquivos

Vá para Arquivos e crie ou escolha a pasta na qual o upload anônimo deve ser feito:

Selecione Link de compartilhamento, Permitir edição, Ocultar lista de arquivos:

Agora você pode enviar o link para a pasta de upload manualmente ou usando a função de envio do Nextcloud, se seu administrador a habilitou.

Fazer upload de arquivos

Usar a função de upload anônimo é simples. Você receberá um link para a pasta de upload, clica no link e, em seguida, verá uma página Nextcloud com um botão “Clique para enviar”:

Isso abre um seletor de arquivos e você seleciona o arquivo ou diretório que deseja carregar. Você também pode simplesmente soltar arquivos na janela.
