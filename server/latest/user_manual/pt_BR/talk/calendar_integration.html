Integração do calendário com o Talk

Reuniões e eventos

Se os eventos do calendário tiverem uma conversa do Talk definida como local do evento, você verá informações sobre os próximos eventos dentro dessa conversa. Dessa forma, você pode se manter informado sobre reuniões ou atividades agendadas diretamente no seu bate-papo. Se o aplicativo Calendário estiver ativado, você pode clicar em um evento para ver os detalhes.

É possível agendar uma reunião diretamente a partir de uma conversa. Na janela de diálogo, você pode definir os detalhes da reunião, como título, data, hora e descrição. Você também pode optar por convidar todos os participantes, incluindo convidados por e-mail, ou selecionar participantes específicos.

Agendar a partir do Calendário

Ao criar um novo evento no Calendário, você pode definir uma conversa do Talk como local do evento. Isso criará uma nova conversa, caso ainda não exista nenhuma.

Quando o evento for criado, você verá um link para a conversa nos detalhes do evento. A conversa também aparecerá na lista de conversas (acessível pelo filtro Eventos).

Assim como as reuniões instantâneas, as conversas de eventos serão excluídas automaticamente após um período de inatividade configurado (por padrão, 28 dias).
