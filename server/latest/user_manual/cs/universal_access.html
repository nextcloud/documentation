Zpřístupnění pro všechny

Universal access is very important to us. We follow web standards and ensure everything is usable with a keyboard and assistive software such as screen readers. We aim to be compliant with the Web Content Accessibility Guidelines 2.1 on AA level, with the high contrast theme even on AAA level. We also follow the German BITV 2.0 guidelines.

Pokud narazíte na jakékoli potíže, neváhejte nám je nahlásit do našeho systému pro správu hlášení. A pokud se chcete zapojit přidejte se do našeho kolektivu návrhářů!

Zvětšování a přizpůsobování zobrazení rozměrům

Webové rozhraní Nextcloud se plně přizpůsobuje rozměrům zobrazení a je tak použitelné na libovolné velikosti obrazovky. Je možné ho přiblížit a oddálit tak, aby vám vyhovovala velikost textu a prvků. Postranní panel pro navigaci i kontextový postranní pruh je možné pro úsporu místa sbalit a rozbalit.

Pohyb po aplikaci prostřednictvím klávesnice

You can navigate the web interface with a keyboard, just as you would with a mouse:

  • Tab and Shift+Tab to move between elements

  • Enter or Space to activate or open the element (depending on the type of element)

  • Escape to be used to close modals, popover menus, and file viewers

  • Left arrow and Right arrow to navigate between photos in the viewer

  • Ctrl+F to focus the search field

  • Ctrl+S to save changes in editors like Nextcloud Text

For quicker navigation, we offer two „skip links“ at the beginning of the document which allow you to:

  • Přeskočit na hlavní obsah

  • Přeskočit do navigace po aplikaci

Nextcloud Talk má klávesové zkratky, které jsou zdokumentované také v nastavení aplikace samotné:

  • C to focus the message input field

  • Escape to unfocus the message input field to be able to use shortcuts

  • F to fullscreen the chat or call

  • V průběhu hovoru:

    • M to toggle the microphone on and off

    • V to toggle video on and off

    • Space for push to talk or push to mute

    • R to raise or lower hand

Nextcloud Mail má také klávesové zkratky, také zdokumentované v nastavení aplikace samotné:

  • C to compose a new message

  • Left arrow to switch to a newer message

  • Right arrow to switch to an older message

  • S to toggle a message as favorite

  • U to toggle a message unread

  • Del to delete a message

  • Ctrl+Enter to send

  • R to refresh and load new mails

Motivy vzhledu, které jsou už součástí

Nabízíme několik motivů vzhledu, které si můžete zapnout a napomoci si tak v zpřístupnění:

  • Motiv vzhledu s vysokým kontrastem: Režim s vysokým kontrastem, který usnadní pohyb po aplikaci. Obrazová kvalita se sníží, ale zřetelnost zvýší.

  • Tmavý motiv vzhledu: Tmavý motiv uleví vašim očím omezením celkové svítivosti a jasu. Pořád ještě ale není zcela hotový, takže pokud narazíte na nějaké problémy, nahlaste je prosím.

  • Písmo pro dyslektiky: OpenDyslexic je volné písmo/font navržené pro zmírnění některých běžných chyb čtení, způsobovaných dyslexií.

K nastavení zpřístupnění se dostanete:

  1. Otevřete nabídku nastavení na konci záhlaví

  2. Vyberte Nastavení

  3. V navigaci vyberte Zpřístupnění

Poznámka

Kontrast prvků se může lišit dle uživatelsky určeného motivu vzhledu. Například hlavní barva motivu je použita pro pozadí záhlaví, přihlašovací stránky a hlavních tlačítek. Pokud toto způsobuje problémy s kontrastem, obraťte se o pomoc na správce vámi využívaného serveru.
