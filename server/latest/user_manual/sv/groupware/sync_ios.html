Synkronisera med iOS

Kalender

  1. Öppna appen Inställningar.

  2. Välj Appar.

  3. Välj Kalender.

  4. Välj Kalenderkonton.

  5. Välj Lägg till konto.

  6. Välj Annat som kontotyp.

  7. Select Add CalDAV account.

  8. För servern anger du serverns domännamn, till exempel example.com.

  9. Ange ditt användarnamn och lösenord.

  10. Välj Nästa.

  11. Öppna Avancerade inställningar

  12. För servern anger du serverns domännamn och sökväg, till exempel example.com/remote.php/dav/principals/users/username/ (ersätt example.com och username med dina värden).

  13. Stäng Avancerade inställningar

Din kalender visas nu i appen Kalender.

Observera

Om du får ett SSL-relaterat felmeddelande kan du prova följande: Se till att du antingen anger både protokollet (https://) och porten (vanligen 443) i fältet Server, till exempel https://example.com:443/remote.php/dav/principals/users/username/, eller inte anger något av dem, som i steg-för-steg-anvisningen ovan. Appen försöker i båda fallen automatiskt använda SSL, vilket du kan kontrollera under ”Avancerade inställningar” för kontot efter att du har sparat det.

Observera

Från och med iOS 12 krävs SSL-kryptering. Inaktivera därför inte SSL (av den anledningen krävs ett certifikat för din domän; https://letsencrypt.org/ fungerar).

Observera

If you select CardDAV only contact syncing will be made available.

Kontakter

  1. Öppna appen Inställningar.

  2. Välj Appar.

  3. Välj Kontakter.

  4. Välj Kontaktkonton.

  5. Välj Lägg till konto.

  6. Välj Annat som kontotyp.

  7. Select Add CardDAV account.

  8. För servern anger du serverns domännamn och sökväg, till exempel example.com/remote.php/dav/principals/users/username/ (ersätt example.com och username med dina värden).

  9. Ange ditt användarnamn och lösenord.

  10. Välj Nästa.

Du bör nu hitta dina kontakter i adressboken på din iPhone.

Observera

Från och med iOS 12 krävs SSL-kryptering. Inaktivera därför inte SSL (av den anledningen krävs ett certifikat för din domän; https://letsencrypt.org/ fungerar).

Observera

If you select CalDAV only calendar syncing will be made available.

Om det fortfarande inte fungerar kan du läsa Troubleshooting Contacts & Calendar eller Troubleshooting Service Discovery.
