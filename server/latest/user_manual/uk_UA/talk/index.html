Розмови

Nextcloud Talk offers audio/video and text chat integrated in Nextcloud. It is available as a web interface, desktop client, and mobile apps for Android and iOS.

You can find out more about Nextcloud Talk on our website. Download the desktop and mobile clients from nextcloud.com/install.
