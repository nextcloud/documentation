Передати право власності

Користувачі можуть передавати право власності на файли та папки іншим користувачам. Спільний доступ до власників перенесених файлів/папки також буде передано.

  1. Перейдіть до *Налаштування *(Праві меню)> *Спільний доступ *.

  2. У розділі *файлів *натисніть кнопку *Виберіть файл або папку, щоб передати *. Відкривається вибір файлу, показуючи всі файли та папки в обліковому записі користувача.

  3. Виберіть файл або папку та натисніть кнопку *Виберіть *. Відобразиться обраний файл або ім’я папки.

  4. Клацніть на * Змінити *, щоб змінити вибір, якщо це необхідно.

  5. Виберіть нового власника, ввівши своє ім’я в поле пошуку поруч із *новим власником *.

  6. Клацніть на *передачу *.

    Примітка

    Автокомпреція або лістинг імені користувача може бути обмежена через конфігурацію адміністративної видимості. Див. Документація адміністратора <https://docs.nextcloud.com/server/latest/admin_manual/configuration_files/file_sharing_configuration.html> _ для детальної інформації.

  7. Цільовий користувач отримує повідомлення, коли їх запитують, чи приймати чи відхиляти вхідну передачу.

  8. Якщо прийнятим, цільовий користувач знаходить передані файли та папки в корені під папкою *, що передається з [користувача] на [timestamp] *.

  9. Користувач джерела повідомляється про прийняття або відхилення шляхом повідомлення.
