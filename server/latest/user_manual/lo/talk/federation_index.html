ເຄືອຂ່າຍຮ່ວມ

ດ້ວຍຄຸນສົມບັດເຄືອຂ່າຍຮ່ວມ, ຜູ້ໃຊ້ສາມາດສ້າງການສົນທະນາຂ້າມລະບົບ Talk ທີ່ເຊື່ອມຕໍ່ກັນ ແລະ ສາມາດນຳໃຊ້ຄຸນສົມບັດຕ່າງໆຂອງ Talk ໄດ້ຄືກັບວ່າຢູ່ໃນເຊີເວີດຽວກັນ.

ຄຸນສົມບັດນີ້ຕ້ອງໄດ້ຮັບການເປີດນຳໃຊ້ໂດຍຜູ້ດູແລລະບົບ.

ການສົ່ງຄຳເຊີນ

ເພື່ອຮັບຄຳເຊີນ, ອີກຝ່າຍຕ້ອງມີ CloudID ຂອງທ່ານ. ທ່ານສາມາດຊອກຫາ CloudID ຂອງທ່ານໄດ້ໃນ ການຕັ້ງຄ່າສ່ວນຕົວ ພາຍໃຕ້ ການແບ່ງປັນ — ມັນຈະມີລັກສະນະຄື user@cloud.example.com.

ຜູ້ດູແລການສົນທະນາສາມາດສົ່ງຄຳເຊີນໄປຫາຜູ້ເຂົ້າຮ່ວມທີ່ຢູ່ເຊີເວີອື່ນໄດ້:

ການຕອບຮັບຄຳເຊີນ

ເມື່ອໄດ້ຮັບການແຈ້ງເຕືອນ, ຜູ້ໃຊ້ຈະເຫັນຕົວເລກສະແດງຈຳນວນຄຳເຊີນທີ່ຄ້າງຢູ່ ເທິງລາຍຊື່ການສົນທະນາ.

ເມື່ອຄລິກໃສ່, ຈະມີຂໍ້ມູນເພີ່ມເຕີມກ່ຽວກັບຜູ້ທີ່ເຊີນມາ, ແລະ ຜູ້ໃຊ້ສາມາດເລືອກທີ່ຈະຍອມຮັບ ຫຼື ປະຕິເສດຄຳເຊີນນັ້ນໄດ້.

ເມື່ອຍອມຮັບຄຳເຊີນແລ້ວ, ການສົນທະນາຈະສະແດງຢູ່ໃນລາຍຊື່ຄືກັນກັບການສົນທະນາອື່ນໆ.

ທ່ານສາມາດນຳໃຊ້ຄຸນສົມບັດນີ້ເພື່ອແຊັດກັບຜູ້ເຂົ້າຮ່ວມຈາກເຊີເວີອື່ນທີ່ເຊື່ອມຕໍ່ກັນ, ເຂົ້າຮ່ວມການໂທ, ແລະ ນຳໃຊ້ຄຸນສົມບັດອື່ນໆທີ່ມີຢູ່ໃນ Talk.
