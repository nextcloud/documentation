ການນຳໃຊ້ຕົວລູກຂ່າຍການຊິງໂຄຣໄນ

ຕົວລູກຂ່າຍ Nextcloud ສຳລັບຄອມພິວເຕີຈະເຮັດວຽກຢູ່ເບື້ອງຫຼັງ ແລະ ປະກົດໃຫ້ເຫັນເປັນໄອຄອນຢູ່ໃນຖາດລະບົບ (Windows, KDE), ແຖບເມນູ (macOS) ຫຼື ເຂດການແຈ້ງເຕືອນ (Linux).

ຕົວບົ່ງບີ້ສະຖານະຈະໃຊ້ໄອຄອນເພື່ອບອກສະຖານະປັດຈຸບັນຂອງການຊິງໂຄຣໄນ. ວົງມົນສີຂຽວທີ່ມີເຄື່ອງໝາຍຖືກສີຂາວບອກໃຫ້ຮູ້ວ່າການຊິງໂຄຣໄນຂອງທ່ານເປັນປັດຈຸບັນແລ້ວ ແລະ ທ່ານກຳລັງເຊື່ອມຕໍ່ກັບເຊີເວີ Nextcloud ຢູ່.

ໄອຄອນສີຟ້າທີ່ມີຮູບເຄິ່ງວົງມົນສີຂາວໝາຍເຖິງການຊິງໂຄຣໄນກຳລັງດຳເນີນການຢູ່.

ໄອຄອນສີເທົາທີ່ມີເສັ້ນຂະໜານບອກໃຫ້ຮູ້ວ່າການຊິງໂຄຣໄນຂອງທ່ານຖືກຢຸດຊົ່ວຄາວ (ສ່ວນຫຼາຍແມ່ນທ່ານເປັນຜູ້ຢຸດເອງ).

ໄອຄອນສີເທົາທີ່ມີສາມຈຸດສີຂາວໝາຍເຖິງຕົວລູກຂ່າຍການຊິງຂອງທ່ານຂາດການເຊື່ອມຕໍ່ກັບເຊີເວີ Nextcloud.

ເມື່ອທ່ານເຫັນວົງມົນສີເຫຼືອງທີ່ມີເຄື່ອງໝາຍ "!" ນັ້ນແມ່ນໄອຄອນຂໍ້ມູນຂ່າວສານ, ທ່ານຄວນຄລິກໃສ່ມັນເພື່ອເບິ່ງລາຍລະອຽດທີ່ຕ້ອງການແຈ້ງໃຫ້ຊາບ.

ວົງມົນສີແດງທີ່ມີເຄື່ອງໝາຍ "x" ສີຂາວ ບົ່ງບອກເຖິງຂໍ້ຜິດພາດໃນການກຳນົດຄ່າ ເຊັ່ນ: ການເຂົ້າສູ່ລະບົບບໍ່ຖືກຕ້ອງ ຫຼື URL ຂອງເຊີເວີບໍ່ຖືກຕ້ອງ.

ໄອຄອນໃນຖາດລະບົບ

ການຄລິກຂວາໃສ່ໄອຄອນໃນຖາດລະບົບຈະເປີດເມນູເພື່ອເຂົ້າເຖິງການເຮັດວຽກຕ່າງໆຢ່າງໄວວາ.

ເມນູນີ້ມີຕົວເລືອກດັ່ງຕໍ່ໄປນີ້:

  • ເປີດໜ້າຕ່າງຫຼັກ

  • ຢຸດການຊິງຊົ່ວຄາວ/ສືບຕໍ່ການຊິງ

  • ການຕັ້ງຄ່າ

  • ອອກຈາກ Nextcloud, ອອກຈາກລະບົບ ແລະ ປິດຕົວລູກຂ່າຍ

ການຄລິກຊ້າຍໃສ່ໄອຄອນໃນຖາດລະບົບຈະເປີດໜ້າຕ່າງຫຼັກຂອງຕົວລູກຂ່າຍ.

ໜ້າຕ່າງຫຼັກຈະສະແດງກິດຈະກຳຫຼ້າສຸດ, ຂໍ້ຜິດພາດ ແລະ ການແຈ້ງເຕືອນຈາກເຊີເວີ.

ເມື່ອຄລິກໃສ່ໜ້າຕ່າງຫຼັກ ແລະ ຄລິກໃສ່ຮູບແທນຕົວ (Avatar) ຂອງຜູ້ໃຊ້, ຈະສາມາດເປີດການຕັ້ງຄ່າໄດ້.

ການກຳນົດຄ່າການຕັ້ງຄ່າບັນຊີ Nextcloud

ຢູ່ທາງເທິງຂອງໜ້າຕ່າງຈະມີແທັບສຳລັບແຕ່ລະບັນຊີທີ່ໄດ້ຕັ້ງຄ່າໄວ້, ແລະ ອີກສອງແທັບສຳລັບການຕັ້ງຄ່າທົ່ວໄປ ແລະ ເຄືອຂ່າຍ. ໃນແທັບບັນຊີຂອງທ່ານຈະມີຟີເຈີດັ່ງນີ້:

  • ສະຖານະການເຊື່ອມຕໍ່, ສະແດງໃຫ້ເຫັນວ່າທ່ານເຊື່ອມຕໍ່ກັບເຊີເວີ Nextcloud ໃດ ແລະ ຊື່ຜູ້ໃຊ້ Nextcloud ຂອງທ່ານ.

  • ພື້ນທີ່ທີ່ໃຊ້ໄປ ແລະ ພື້ນທີ່ວ່າງເທິງເຊີເວີ.

  • ສະຖານະການຊິງໂຄຣໄນປັດຈຸບັນ.

  • ປຸ່ມ ເພີ່ມການເຊື່ອມຕໍ່ການຊິງໂຟນເດີ.

ປຸ່ມນ້ອຍໆທີ່ມີສາມຈຸດ (ເມນູເພີ່ມເຕີມ) ທີ່ຢູ່ທາງເບື້ອງຂວາຂອງແຖບສະຖານະການຊິງ ຈະມີຕົວເລືອກເພີ່ມເຕີມ:

  • ເປີດໂຟນເດີ

  • ເລືອກສິ່ງທີ່ຈະຊິງ (ຕົວເລືອກນີ້ຈະປະກົດຂຶ້ນເມື່ອໂຄງສ້າງໄຟລ໌ຂອງທ່ານຖືກຫຍໍ້ໄວ້ ແລະ ຈະຂະຫຍາຍໂຄງສ້າງໄຟລ໌ອອກ)

  • ຢຸດການຊິງຊົ່ວຄາວ / ສືບຕໍ່ການຊິງ

  • ລົບການເຊື່ອມຕໍ່ການຊິງໂຟນເດີ

  • ຄວາມພ້ອມໃຊ້ງານ (ໃຊ້ໄດ້ສະເພາະເມື່ອເປີດໃຊ້ການຮອງຮັບໄຟລ໌ສະເໝືອນ)

  • ເປີດໃຊ້ການຮອງຮັບໄຟລ໌ສະເໝືອນ/ປິດໃຊ້ການຮອງຮັບໄຟລ໌ສະເໝືອນ

ເປີດໂຟນເດີ ຈະເປີດໂຟນເດີຊິງ Nextcloud ໃນເຄື່ອງຂອງທ່ານ.

ຢຸດການຊິງຊົ່ວຄາວ ຈະຢຸດການເຮັດວຽກຂອງການຊິງໂດຍບໍ່ມີການປ່ຽນແປງໃດໆກັບບັນຊີຂອງທ່ານ. ມັນຈະສືບຕໍ່ອັບເດດລາຍຊື່ໄຟລ໌ ແລະ ໂຟນເດີ ໂດຍບໍ່ມີການດາວໂຫຼດ ຫຼື ອັບເດດໄຟລ໌. ເພື່ອຢຸດກິດຈະກຳການຊິງທັງໝົດໃຫ້ໃຊ້ ລົບການເຊື່ອມຕໍ່ການຊິງໂຟນເດີ.

Note

Nextcloud ຈະບໍ່ເກັບຮັກສາ mtime (ເວລາທີ່ມີການແກ້ໄຂ) ຂອງໄດເລັກທໍຣີ, ແຕ່ມັນຈະອັບເດດ mtime ໃນໄຟລ໌. ເບິ່ງ Wrong folder date when syncing ສຳລັບການສົນທະນາກ່ຽວກັບເລື່ອງນີ້.

ການເພີ່ມບັນຊີໃໝ່

ທ່ານສາມາດຕັ້ງຄ່າບັນຊີ Nextcloud ໄດ້ຫຼາຍບັນຊີໃນຕົວລູກຂ່າຍການຊິງ. ພຽງແຕ່ຄລິກທີ່ປຸ່ມ ບັນຊີ > ເພີ່ມໃໝ່ ໃນແທັບບັນຊີໃດໜຶ່ງເພື່ອເພີ່ມບັນຊີໃໝ່, ຈາກນັ້ນເຮັດຕາມຂັ້ນຕອນການສ້າງບັນຊີ. ບັນຊີໃໝ່ຈະປະກົດເປັນແທັບໃໝ່ໃນໜ້າຕ່າງການຕັ້ງຄ່າ, ເຊິ່ງທ່ານສາມາດປັບແຕ່ງການຕັ້ງຄ່າໄດ້ຕະຫຼອດເວລາ. ໃຊ້ ບັນຊີ > ລົບອອກ ເພື່ອລົບບັນຊີ.

ໄອຄອນຊ້ອນທັບໃນຕົວຈັດການໄຟລ໌

ຕົວລູກຂ່າຍການຊິງ Nextcloud ຈະມີໄອຄອນຊ້ອນທັບ ເພີ່ມເຕີມຈາກໄອຄອນປະເພດໄຟລ໌ປົກກະຕິ ສຳລັບຕົວຈັດການໄຟລ໌ໃນລະບົບຂອງທ່ານ (Explorer ໃນ Windows, Finder ໃນ Mac ແລະ Nautilus ໃນ Linux) ເພື່ອບົ່ງບອກສະຖານະການຊິງຂອງໄຟລ໌ Nextcloud ຂອງທ່ານ.

ໄອຄອນຊ້ອນທັບຈະຄ້າຍຄືກັບໄອຄອນໃນຖາດລະບົບທີ່ກ່າວມາຂ້າງເທິງ. ພວກມັນຈະສະແດງຜົນແຕກຕ່າງກັນໃນໄຟລ໌ ແລະ ໄດເລັກທໍຣີ ຕາມສະຖານະການຊິງ ແລະ ຂໍ້ຜິດພາດ.

ໄອຄອນຊ້ອນທັບຂອງແຕ່ລະໄຟລ໌ຈະບົ່ງບອກສະຖານະການຊິງປັດຈຸບັນຂອງມັນ. ຖ້າໄຟລ໌ຖືກຊິງກົງກັບເວີຊັນໃນເຊີເວີແລ້ວ, ມັນຈະສະແດງເຄື່ອງໝາຍຖືກສີຂຽວ.

ຖ້າໄຟລ໌ຖືກລະເວັ້ນຈາກການຊິງ, ຕົວຢ່າງເຊັ່ນ ຍ້ອນວ່າມັນຢູ່ໃນລາຍຊື່ທີ່ຍົກເວັ້ນ ຫຼື ຍ້ອນວ່າມັນເປັນ symbolic link, ມັນຈະສະແດງໄອຄອນເຕືອນ.

ຖ້າມີຂໍ້ຜິດພາດໃນການຊິງ ຫຼື ໄຟລ໌ຖືກຕິດບັນຊີດຳ, ມັນຈະສະແດງເຄື່ອງໝາຍ X ສີແດງທີ່ເຫັນໄດ້ແຈ້ງ.

ຖ້າໄຟລ໌ກຳລັງລໍຖ້າການຊິງ ຫຼື ກຳລັງຊິງຢູ່, ໄອຄອນຊ້ອນທັບຈະສະແດງເປັນຮູບລູກສອນໝູນວຽນສີຟ້າ.

ເມື່ອຕົວລູກຂ່າຍອອບລາຍ, ຈະບໍ່ມີໄອຄອນໃດໆສະແດງຂຶ້ນ ເພື່ອບອກໃຫ້ຮູ້ວ່າໂຟນເດີບໍ່ໄດ້ຢູ່ໃນການຊິງ ແລະ ບໍ່ມີການປ່ຽນແປງໃດໆຖືກຊິງໄປຫາເຊີເວີ.

ໄອຄອນຊ້ອນທັບຂອງໄດເລັກທໍຣີທີ່ຊິງແລ້ວຈະບົ່ງບອກສະຖານະຂອງໄຟລ໌ຕ່າງໆໃນໄດເລັກທໍຣີນັ້ນ. ຖ້າມີຂໍ້ຜິດພາດໃນການຊິງ, ໄດເລັກທໍຣີຈະຖືກໝາຍດ້ວຍໄອຄອນເຕືອນ.

ຖ້າໄດເລັກທໍຣີມີໄຟລ໌ທີ່ຖືກລະເວັ້ນເຊິ່ງຖືກໝາຍດ້ວຍໄອຄອນເຕືອນ, ສິ່ງນີ້ຈະບໍ່ມີຜົນຕໍ່ສະຖານະຂອງໄດເລັກທໍຣີຫຼັກ (Parent directory).

ຕັ້ງສະຖານະຜູ້ໃຊ້

ຖ້າທ່ານໄດ້ຕິດຕັ້ງແອັບສະຖານະຜູ້ໃຊ້ໃນເຊີເວີ Nextcloud ຂອງທ່ານ, ທ່ານສາມາດຕັ້ງສະຖານະຜູ້ໃຊ້ໄດ້ຈາກຕົວລູກຂ່າຍຄອມພິວເຕີ. ໂດຍການເປີດໜ້າຕ່າງຫຼັກ, ຈາກນັ້ນຄລິກທີ່ຮູບແທນຕົວຂອງທ່ານ ແລະ ຄລິກທີ່ສາມຈຸດ. ໃນເມນູທີ່ເປີດຂຶ້ນໃຫ້ຄລິກທີ່ ຕັ້ງສະຖານະ.

ໃນໜ້າຕ່າງທີ່ເປີດຂຶ້ນ, ທ່ານສາມາດຕັ້ງສະຖານະອອນລາຍໄດ້ໂດຍການຄລິກເລືອກ ອອນລາຍ, ບໍ່ຢູ່, ຫ້າມລົບກວນ ຫຼື ເຊື່ອງຕົວ. ທ່ານຍັງສາມາດຕັ້ງຂໍ້ຄວາມສະຖານະເອງໄດ້ໃນຊ່ອງໃສ່ຂໍ້ຄວາມ ຫຼື ເລືອກຂໍ້ຄວາມສະຖານະທີ່ມີໄວ້ໃຫ້ແລ້ວ. ນອກຈາກນີ້ຍັງສາມາດຕັ້ງອີໂມຈິໄດ້ໂດຍການຄລິກປຸ່ມອີໂມຈິຂ້າງຊ່ອງໃສ່ຂໍ້ຄວາມ. ສິ່ງສຸດທ້າຍທີ່ທ່ານອາດຈະຕັ້ງຄ່າແມ່ນເວລາທີ່ຈະໃຫ້ລ້າງສະຖານະ. ທ່ານສາມາດເລືອກໄລຍະເວລາທີ່ຈະໃຫ້ລ້າງສະຖານະໄດ້ໂດຍການຄລິກປຸ່ມທາງເບື້ອງຊ້າຍຂອງຂໍ້ຄວາມ ລ້າງຂໍ້ຄວາມສະຖານະຫຼັງຈາກ.

ຖ້າທ່ານພໍໃຈກັບສະຖານະທີ່ສ້າງແລ້ວ, ທ່ານສາມາດເປີດໃຊ້ສະຖານະນີ້ໄດ້ດ້ວຍປຸ່ມ ຕັ້ງຂໍ້ຄວາມສະຖານະ. ຖ້າທ່ານມີສະຖານະທີ່ຕັ້ງໄວ້ກ່ອນແລ້ວ, ທ່ານສາມາດລ້າງສະຖານະໄດ້ໂດຍການຄລິກປຸ່ມ ລ້າງຂໍ້ຄວາມສະຖານະ.

ການແບ່ງປັນຈາກຄອມພິວເຕີຂອງທ່ານ

ຕົວລູກຂ່າຍການຊິງ Nextcloud ເຮັດວຽກຮ່ວມກັບຕົວຈັດການໄຟລ໌ຂອງທ່ານ. Finder ໃນ macOS ແລະ Explorer ໃນ Windows. ສຳລັບຜູ້ໃຊ້ Linux ຕ້ອງຕິດຕັ້ງແພັກເກັດເພີ່ມເຕີມຕາມຕົວຈັດການໄຟລ໌ທີ່ໃຊ້ ເຊັ່ນ: nautilus-nextcloud (Ubuntu/Debian), dolphin-nextcloud (Kubuntu), nemo-nextcloud ແລະ caja-nextcloud. ທ່ານສາມາດສ້າງລິ້ງແບ່ງປັນ ແລະ ແບ່ງປັນກັບຜູ້ໃຊ້ Nextcloud ພາຍໃນໄດ້ຄືກັນກັບການໃຊ້ຜ່ານເວັບໄຊ Nextcloud.

ໃນຕົວຈັດການໄຟລ໌ຂອງທ່ານ, ໃຫ້ຄລິກຂວາໃສ່ໄຟລ໌ ແລະ ໃນເມນູໃຫ້ໄປທີ່ Nextcloud ຈາກນັ້ນຄລິກທີ່ ຕົວເລືອກການແບ່ງປັນ ເພື່ອເປີດໜ້າຕ່າງການແບ່ງປັນ.

ຈາກໜ້າຕ່າງນີ້ ທ່ານສາມາດແບ່ງປັນໄຟລ໌ໄດ້.

ໜ້າຕ່າງທົ່ວໄປ

ໜ້າຕ່າງທົ່ວໄປມີຕົວເລືອກການຕັ້ງຄ່າ ເຊັ່ນ ເລີ່ມວຽກເມື່ອເປີດເຄື່ອງ, ໃຊ້ໄອຄອນສີດຽວ, ແລະ ສະແດງການແຈ້ງເຕືອນຈາກເຊີເວີ. ນີ້ແມ່ນບ່ອນທີ່ທ່ານຈະພົບເຫັນປຸ່ມ ແກ້ໄຂໄຟລ໌ທີ່ຖືກລະເວັ້ນ ເພື່ອເປີດຕົວແກ້ໄຂລາຍຊື່ໄຟລ໌ທີ່ຖືກລະເວັ້ນ, ແລະ ຂໍການຢືນຢັນກ່ອນດາວໂຫຼດໂຟນເດີທີ່ມີຂະໜາດໃຫຍ່ກວ່າ [ຂະໜາດໂຟນເດີ].

ການນຳໃຊ້ໜ້າຕ່າງເຄືອຂ່າຍ

ໜ້າຕ່າງການຕັ້ງຄ່າເຄືອຂ່າຍຊ່ວຍໃຫ້ທ່ານກຳນົດຄ່າພຣັອກຊີ (Proxy) ແລະ ຈຳກັດແບນວິດໃນການດາວໂຫຼດ ແລະ ອັບໂຫຼດ.

ການນຳໃຊ້ຕົວແກ້ໄຂໄຟລ໌ທີ່ຖືກລະເວັ້ນ

ທ່ານອາດຈະມີບາງໄຟລ໌ ຫຼື ໄດເລັກທໍຣີໃນເຄື່ອງທີ່ບໍ່ຕ້ອງການສຳຮອງຂໍ້ມູນ ແລະ ເກັບໄວ້ໃນເຊີເວີ. ເພື່ອລະບຸ ແລະ ຍົກເວັ້ນໄຟລ໌ ຫຼື ໄດເລັກທໍຣີເຫຼົ່ານີ້, ທ່ານສາມາດໃຊ້ ຕົວແກ້ໄຂໄຟລ໌ທີ່ຖືກລະເວັ້ນ (ໃນແທັບທົ່ວໄປ).

ເພື່ອຄວາມສະດວກ, ຕົວແກ້ໄຂໄດ້ມີລາຍຊື່ຮູບແບບການລະເວັ້ນມາດຕະຖານໄວ້ໃຫ້ແລ້ວ. ຮູບແບບເຫຼົ່ານີ້ຖືກເກັບໄວ້ໃນໄຟລ໌ລະບົບ (ປົກກະຕິແມ່ນ sync-exclude.lst) ທີ່ຢູ່ໃນໂຟນເດີຕິດຕັ້ງແອັບ Nextcloud. ທ່ານບໍ່ສາມາດແກ້ໄຂຮູບແບບທີ່ມີມາໃຫ້ເບື້ອງຕົ້ນນີ້ໄດ້ໂດຍກົງຈາກຕົວແກ້ໄຂ. ແນວໃດກໍຕາມ ຖ້າຈຳເປັນ, ທ່ານສາມາດເອົາເມົ້າໄປຊີ້ໃສ່ຮູບແບບໃດໜຶ່ງໃນລາຍຊື່ເພື່ອເບິ່ງທີ່ຢູ່ໄຟລ໌, ຈາກນັ້ນໄປຫາໄຟລ໌ນັ້ນ ແລະ ແກ້ໄຂໄຟລ໌ sync-exclude.lst ໄດ້.

Note

ການແກ້ໄຂໄຟລ໌ກຳນົດການຍົກເວັ້ນລະດັບລະບົບອາດຈະເຮັດໃຫ້ຕົວລູກຂ່າຍໃຊ້ງານບໍ່ໄດ້ ຫຼື ເຮັດວຽກຜິດພາດ.

ແຕ່ລະແຖວໃນຕົວແກ້ໄຂປະກອບດ້ວຍຂໍ້ຄວາມຮູບແບບການລະເວັ້ນ. ໃນການສ້າງຮູບແບບເອງ, ນອກຈາກການໃຊ້ຕົວອັກສອນປົກກະຕິແລ້ວ, ທ່ານຍັງສາມາດໃຊ້ຕົວອັກສອນພິເສດ (wildcards) ເພື່ອຈັບຄູ່ຂໍ້ມູນ. ຕົວຢ່າງ: ທ່ານສາມາດໃຊ້ດອກຈັນ (*) ແທນຕົວອັກສອນຈຳນວນເທົ່າໃດກໍໄດ້ ຫຼື ເຄື່ອງໝາຍຖາມ (?) ແທນຕົວອັກສອນຕົວດຽວ.

ຮູບແບບທີ່ລົງທ້າຍດ້ວຍເຄື່ອງໝາຍສະແລັຊ (/) ຈະຖືກນຳໃຊ້ສະເພາະກັບໄດເລັກທໍຣີໃນເສັ້ນທາງທີ່ກຳລັງກວດສອບເທົ່ານັ້ນ.

Note

ລາຍການທີ່ເພີ່ມເອງໃນປັດຈຸບັນຍັງບໍ່ໄດ້ຖືກກວດສອບຄວາມຖືກຕ້ອງຂອງໄວຍາກອນໂດຍຕົວແກ້ໄຂ, ດັ່ງນັ້ນທ່ານຈະບໍ່ເຫັນການເຕືອນຖ້າຂຽນຜິດ. ຖ້າການຊິງຂອງທ່ານບໍ່ເຮັດວຽກຕາມທີ່ຄາດໄວ້, ໃຫ້ກວດສອບໄວຍາກອນຂອງທ່ານຄືນ.

ແຕ່ລະຮູບແບບໃນລາຍຊື່ຈະມີກ່ອງເຄື່ອງໝາຍຢູ່ທາງຫຼັງ. ເມື່ອກ່ອງຖືກໝາຍຖືກ, ນອກຈາກຈະລະເວັ້ນໄຟລ໌ ຫຼື ໄດເລັກທໍຣີທີ່ກົງກັບຮູບແບບແລ້ວ, ໄຟລ໌ໃດທີ່ກົງກັນຈະຖືກຖືວ່າເປັນ "ເມຕາເດຕາຊົ່ວຄາວ" (fleeting metadata) ແລະ ຈະຖືກລົບອອກໂດຍຕົວລູກຂ່າຍ.

ນອກຈາກການຍົກເວັ້ນໄຟລ໌ ແລະ ໄດເລັກທໍຣີທີ່ໃຊ້ຮູບແບບທີ່ກຳນົດໃນລາຍຊື່ນີ້ແລ້ວ:

  • ຕົວລູກຂ່າຍ Nextcloud ຈະຍົກເວັ້ນໄຟລ໌ທີ່ມີຕົວອັກສອນທີ່ບໍ່ສາມາດຊິງໄປຫາລະບົບໄຟລ໌ອື່ນໆໄດ້ສະເໝີ.

  • ໄຟລ໌ທີ່ເຮັດໃຫ້ເກີດຂໍ້ຜິດພາດສະເພາະຕົວສາມຄັ້ງໃນລະຫວ່າງການຊິງຈະຖືກລົບອອກ. ແນວໃດກໍຕາມ, ຕົວລູກຂ່າຍຍັງມີຕົວເລືອກໃຫ້ລອງຊິງໃໝ່ອີກສາມຄັ້ງສຳລັບໄຟລ໌ທີ່ເກີດຂໍ້ຜິດພາດ.
