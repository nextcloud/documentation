ແອັບເມລຖືກຕິດຕັ້ງມາພ້ອມກັບ Nextcloud Hub ໂດຍມາດຕະຖານ, ແຕ່ສາມາດປິດການໃຊ້ງານໄດ້. ກະລຸນາສອບຖາມຜູ້ດູແລລະບົບຂອງທ່ານ.

  2. ເລືອກລະຫວ່າງ ແບບລາຍການ, ແບບແບ່ງລວງຕັ້ງ ແລະ ແບບແບ່ງລວງນອນ

ໃຊ້ໂໝດກະທັດຮັດ

Added in version 5.7.0.

ໂໝດກະທັດຮັດຊ່ວຍໃຫ້ເບິ່ງຂໍ້ຄວາມໄດ້ສະອາດຕາ ແລະ ມີປະສິດທິພາບຫຼາຍຂຶ້ນ. ຮູບຕົວແທນຈະຖືກເຊື່ອງໄວ້, ຊ່ອງໝາຍເລືອກຈະສະແດງໃຫ້ເຫັນຕະຫຼອດ, ແລະ ການເບິ່ງຕົວຢ່າງຂໍ້ຄວາມຈະຖືກຍົກອອກ. ມັນຊ່ວຍປະຢັດເນື້ອທີ່ ເຮັດໃຫ້ທ່ານເຫັນອີເມລໄດ້ຫຼາຍຂຶ້ນໃນບາດດຽວ.

  1. ໄປທີ່ການຕັ້ງຄ່າເມລ

  2. ໄປທີ່ ຮູບລັກສະນະ

  3. ເປີດ/ປິດ ໂໝດກະທັດຮັດ

ການສະແດງຂໍ້ຄວາມ / ໂໝດການເຮັດວຽກ

Added in version 5.2.

ເມລສາມາດສະຫຼັບລະຫວ່າງສອງຮູບແບບການເບິ່ງ ແລະ ການເຮັດວຽກທີ່ແຕກຕ່າງກັນຄື: ແບບຕໍ່ເນື່ອງ (Threaded) ແລະ ແບບດ່ຽວ (Singleton).

ໃນໂໝດ ແບບຕໍ່ເນື່ອງ (Threaded), ຂໍ້ຄວາມຈະຖືກຈັດກຸ່ມຕາມການສົນທະນາ. ໃນລາຍການຂໍ້ຄວາມ, ຂໍ້ຄວາມທີ່ກ່ຽວຂ້ອງກັນຈະຖືກຊ້ອນກັນໄວ້ເພື່ອໃຫ້ສະແດງສະເພາະຂໍ້ຄວາມຫຼ້າສຸດເທົ່ານັ້ນ, ແລະ ຂໍ້ຄວາມທີ່ກ່ຽວຂ້ອງທັງໝົດຈະສະແດງຢູ່ໃນແຜງສະແດງຂໍ້ຄວາມຫຼັງຈາກທີ່ເລືອກກຸ່ມຂໍ້ຄວາມນັ້ນ. ໂໝດນີ້ມີປະໂຫຍດໃນການຕິດຕາມການສົນທະນາ ແລະ ເຂົ້າໃຈບໍລິບົດຂອງການຕອບກັບ. ໃນໂໝດນີ້, ການຈັດການຂໍ້ຄວາມເຊັ່ນ ການຍ້າຍ ແລະ ການລຶບ ຈະມີຜົນຕໍ່ທັງໝົດກຸ່ມການສົນທະນາ.

ໃນໂໝດ ແບບດ່ຽວ (Singleton), ຂໍ້ຄວາມຈະຖືກສະແດງແຍກກັນເປັນແຕ່ລະສະບັບ ທັງໃນລາຍການຂໍ້ຄວາມ ແລະ ແຜງສະແດງຂໍ້ຄວາມ. ການຈັດການເຊັ່ນ ການຍ້າຍ ຫຼື ການລຶບ ຈະມີຜົນສະເພາະຂໍ້ຄວາມທີ່ເລືອກເທົ່ານັ້ນ. ໂໝດນີ້ມີປະໂຫຍດເມື່ອທ່ານຕ້ອງການຈັດການຂໍ້ຄວາມແຍກກັນ ໂດຍບໍ່ໃຫ້ມີຜົນຕໍ່ການສົນທະນາທັງໝົດ.

  1. ໄປທີ່ການຕັ້ງຄ່າເມລ

  2. ເລືອກລະຫວ່າງ ແບບຕໍ່ເນື່ອງ, ແບບດ່ຽວ

  1. ເປີດໃຊ້ງານແອັບເມລຈາກສ່ວນແອັບພລິເຄຊັນ

ກ່ອງຈົດໝາຍບຸລິມະສິດມີ 2 ສ່ວນຄື ສຳຄັນ ແລະ ອື່ນໆ. ຂໍ້ຄວາມຈະຖືກໝາຍວ່າສຳຄັນໂດຍອັດຕະໂນມັດ ອີງຕາມຂໍ້ຄວາມທີ່ທ່ານມີປະຕິກິລິຍານຳ ຫຼື ໝາຍວ່າສຳຄັນ. ໃນໄລຍະທຳອິດ ທ່ານອາດຈະຕ້ອງໄດ້ປ່ຽນລະດັບຄວາມສຳຄັນດ້ວຍຕົນເອງເພື່ອສອນລະບົບ, ແຕ່ມັນຈະດີຂຶ້ນຕາມການເວລາ.

ການແຍກປະເພດອັດຕະໂນມັດນີ້ເປັນທາງເລືອກ. ທ່ານສາມາດເລືອກບໍ່ໃຊ້ໄດ້ໃນຕອນຕັ້ງຄ່າບັນຊີ. ນອກຈາກນີ້ ຍັງສາມາດເປີດ ຫຼື ປິດການແຍກປະເພດໄດ້ໃນການຕັ້ງຄ່າບັນຊີໄດ້ທຸກເວລາ.

ທຸກກ່ອງຈົດໝາຍ

  6. ..ແລະ ອື່ນໆ

ສາມາດເບິ່ງໄດ້ໃນເມນູການຈັດການຂອງບັນຊີເມລ. ຢູ່ທີ່ນັ້ນທ່ານສາມາດແກ້ໄຂ, ເພີ່ມ ຫຼື ລຶບການຕັ້ງຄ່າຕ່າງໆໄດ້ຕາມຄວາມຕ້ອງການ.

ການຄົ້ນຫາແບບລວມສູນ

ແອັບເມລເຮັດວຽກຮ່ວມກັບລະບົບ ການຄົ້ນຫາແບບລວມສູນ ຂອງ Nextcloud (ເບິ່ງລາຍລະອຽດທີ່ ສ່ວນຕິດຕໍ່ຜູ້ໃຊ້ຜ່ານເວັບຂອງ Nextcloud). ທ່ານສາມາດຄົ້ນຫາອີເມລຈາກທຸກບັນຊີຂອງທ່ານໄດ້ ໂດຍໃຊ້ແຖບຄົ້ນຫາຢູ່ສ່ວນຫົວຂອງ Nextcloud.

ການຄົ້ນຫາຈະກວດສອບຈາກຫົວຂໍ້ເມລ ແລະ ຊ່ອງຜູ້ສົ່ງ/ຜູ້ຮັບ. ຫາກຕ້ອງການຄົ້ນຫາໃນເນື້ອໃນອີເມລ, ໃຫ້ໃຊ້ຄຸນສົມບັດການຄົ້ນຫາພາຍໃນແອັບ.

ຄົ້ນຫາໃນກ່ອງຈົດໝາຍ

  1. ການມອບໝາຍສິດນີ້ຕ້ອງໄດ້ຮັບການຕັ້ງຄ່າໃນເຊີເວີເມລໂດຍຜູ້ດູແລລະບົບ

ອີເມລທີ່ສົ່ງໄປແລ້ວອາດຈະບໍ່ສະແດງໃຫ້ບັນຊີຕົ້ນສະບັບເຫັນ ຫາກມັນຖືກເກັບໄວ້ໃນກ່ອງຈົດໝາຍ ສົ່ງແລ້ວ (Sent) ສ່ວນຕົວຂອງທ່ານ.

ກົດປຸ່ມປິດໃນໜ້າຕ່າງ ຫຼື ທີ່ແຖບສະແດງສະຖານະມຸມຂວາລຸ່ມເພື່ອຢຸດການແກ້ໄຂຂໍ້ຄວາມ. ຮ່າງຈົດໝາຍຈະຖືກບັນທຶກໄວ້ໃນກ່ອງຮ່າງຈົດໝາຍຂອງທ່ານໂດຍອັດຕະໂນມັດ.

ຂໍ້ມູນຜູ້ຮັບໃນໜ້າຂຽນຂໍ້ຄວາມ

Added in version 4.2.

ເມື່ອທ່ານເພີ່ມຜູ້ຮັບ ຫຼື ຜູ້ຕິດຕໍ່ຄົນທຳອິດໃນຊ່ອງ "ເຖິງ (To)", ແຜງດ້ານຂວາຈະປະກົດຂຶ້ນເພື່ອສະແດງລາຍລະອຽດໂປຣໄຟລ໌ຂອງຜູ້ຕິດຕໍ່ນັ້ນ. ການເພີ່ມຜູ້ຕິດຕໍ່ຄົນທີສອງຈະເຮັດໃຫ້ລາຍຊື່ຖືກຫຍໍ້ລົງ, ເຊິ່ງທ່ານສາມາດເລືອກ ແລະ ຂະຫຍາຍເບິ່ງລາຍລະອຽດຂອງຜູ້ຕິດຕໍ່ແຕ່ລະຄົນໄດ້. ຫາກທ່ານຕ້ອງການມີສະມາທິກັບການຂຽນເທົ່ານັ້ນ, ທ່ານສາມາດເຊື່ອງແຜງດ້ານຂວາໄດ້ໂດຍການກົດໄອຄອນຂະຫຍາຍຢູ່ມຸມຂວາເທິງ. ແລະ ເພື່ອໃຫ້ສະແດງແຜງດ້ານຂວາຄືນໃໝ່, ພຽງແຕ່ກົດໄອຄອນຫຍໍ້ຢູ່ບ່ອນເກົ່ານັ້ນ.

ກ່ອງຂາອອກ (Outbox)

ເມື່ອຂຽນຂໍ້ຄວາມແລ້ວ ແລະ ກົດປຸ່ມ "ສົ່ງ", ຂໍ້ຄວາມຈະຖືກເພີ່ມເຂົ້າໃນກ່ອງຂາອອກ ເຊິ່ງສາມາດເບິ່ງໄດ້ຢູ່ມຸມຊ້າຍລຸ່ມຂອງແຖບດ້ານຂ້າງ.

ທ່ານຍັງສາມາດຕັ້ງວັນທີ ແລະ ເວລາເພື່ອສົ່ງໃນອະນາຄົດໄດ້ (ເບິ່ງ ການຕັ້ງເວລາສົ່ງຂໍ້ຄວາມ) - ຂໍ້ຄວາມຈະຖືກເກັບໄວ້ໃນກ່ອງຂາອອກຈົນກວ່າຈະຮອດວັນທີ ແລະ ເວລາທີ່ທ່ານເລືອກ, ຫຼັງຈາກນັ້ນມັນຈະຖືກສົ່ງໂດຍອັດຕະໂນມັດ.

ກ່ອງຂາອອກຈະສະແດງໃຫ້ເຫັນກໍຕໍ່ເມື່ອມີຂໍ້ຄວາມທີ່ກຳລັງລໍຖ້າການດຳເນີນການເທົ່ານັ້ນ.

ທ່ານສາມາດເປີດໜ້າຂຽນຂໍ້ຄວາມເພື່ອແກ້ໄຂເມລໃນກ່ອງຂາອອກໄດ້ທຸກເວລາ ກ່ອນທີ່ການສົ່ງຈະເລີ່ມຕົ້ນຂຶ້ນ.

Note

ເມື່ອເກີດຂໍ້ຜິດພາດໃນລະຫວ່າງການສົ່ງ, ອາດມີຂໍ້ຄວາມແຈ້ງເຕືອນ 3 ຮູບແບບຄື:

ອີເມລຖືກສົ່ງໄປແລ້ວ ແຕ່ບໍ່ສາມາດສຳເນົາໄວ້ໃນກ່ອງ "ສົ່ງແລ້ວ". ຂໍ້ຜິດພາດນີ້ຈະຖືກຈັດການໂດຍກ່ອງຂາອອກ ແລະ ຈະພະຍາຍາມສຳເນົາຄືນໃໝ່.

ການສົ່ງອາດຈະລົ້ມເຫຼວ ຫຼື ບໍ່ກໍໄດ້. ເຊີເວີເມລບໍ່ສາມາດລະບຸສະຖານະຂອງຂໍ້ຄວາມໄດ້. ເນື່ອງຈາກແອັບເມລບໍ່ມີທາງຮູ້ສະຖານະ (ວ່າສົ່ງແລ້ວ ຫຼື ຍັງ), ຂໍ້ຄວາມຈະຍັງຄົງຢູ່ໃນກ່ອງຂາອອກ ແລະ ຜູ້ໃຊ້ຕ້ອງເປັນຜູ້ຕັດສິນໃຈເອງວ່າຈະດຳເນີນການແນວໃດຕໍ່.

  • ກົງກັບຮູບແບບ: ການໃຊ້ເຄື່ອງໝາຍຊ່ວຍຄົ້ນຫາ. ເຄື່ອງໝາຍ "*" ແທນຕົວອັກສອນໃດໆກໍໄດ້ບໍ່ຈຳກັດຈຳນວນ, ສ່ວນ "?" ແທນຕົວອັກສອນດຽວ. ຕົວຢ່າງ: "report" ຈະກົງກັບ "Business report 2024".

  • ຕອບຮັບແບບບໍ່ແນ່ນອນ

  1. ເປີດຂໍ້ຄວາມທີ່ມີຄຳເຊີນປະຊຸມ

  4. ກົດ "ນຳເຂົ້າສູ່ປະຕິທິນ" ແລ້ວເລືອກປະຕິທິນທີ່ຕ້ອງການ.

  3. ເປີດໃຊ້ງານ ສ້າງການນັດໝາຍແບບບໍ່ແນ່ນອນໃນປະຕິທິນໂດຍອັດຕະໂນມັດ

Note

ເມື່ອເປີດການຕັ້ງຄ່ານີ້, ຄຳເຊີນຈະຍັງປະກົດຢູ່ໃນລາຍການເມລຂອງທ່ານ, ແຕ່ພວກມັນຈະຖືກເພີ່ມເຂົ້າໃນປະຕິທິນໂດຍອັດຕະໂນມັດ.

ທາງລັດຄີບອດ

ແອັບເມລມີທາງລັດຄີບອດຫຼາຍຢ່າງເພື່ອຊ່ວຍໃຫ້ທ່ານໃຊ້ງານໄດ້ໄວຂຶ້ນ.

ສຳລັບລາຍຊື່ທາງລັດທັງໝົດທີ່ຮອງຮັບ, ສາມາດເບິ່ງໄດ້ທີ່ການຕັ້ງຄ່າເມລໃນລະບົບຂອງທ່ານ.
