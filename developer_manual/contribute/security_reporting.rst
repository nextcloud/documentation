===================
Security Reporting
===================

.. note::
   This page is under development.

This page describes the responsible disclosure process for reporting security vulnerabilities to the Nextcloud security team.

.. note::
   This is **not** the same as the secure coding guidelines for app developers, which have moved to :doc:`/app_development/php/backend/security/secure_coding`.

This page will cover:

- How to contact the Nextcloud security team
- What information to include in a security report
- The disclosure timeline and process
- The Nextcloud security bug bounty program
