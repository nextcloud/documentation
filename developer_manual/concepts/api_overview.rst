==============================
Introduction to Nextcloud APIs
==============================

.. note::
   This page is under development.

Nextcloud provides several APIs for different development contexts:

- **OCP (PHP Public API):** Stable PHP interfaces for apps running inside the Nextcloud server.
- **OCS (REST API):** HTTP-based API for external clients, integrations, and scripts.
- **WebDAV:** HTTP protocol for file operations — uploading, downloading, listing, searching, and managing files.
- **JavaScript APIs:** Browser-side npm packages for building Nextcloud app front-ends.

For detailed API documentation, see :doc:`/api_reference/index`.

This page will provide a conceptual overview of how these APIs relate to each other and when to use each one.
