================================
Welcome to Nextcloud Development
================================

Welcome to the Nextcloud Developer Manual. This documentation covers everything you need to build apps, integrations, and clients for the Nextcloud platform.

Whether you are building a PHP app that runs inside Nextcloud, a Docker-based ExApp, a mobile or desktop client, or contributing to the Nextcloud server itself, this manual is your starting point.

**Where to begin:**

- New to Nextcloud development? Start with :doc:`/concepts/index` to understand how the platform works.
- Ready to build? Head to :doc:`/app_development/index` for app development guides.
- Looking for API documentation? See :doc:`/api_reference/index`.
- Want to contribute to Nextcloud itself? Visit :doc:`/contribute/index`.
